# 实战交易手册

> 基于「柚子小杰杰」短线交易体系提炼的精简实战手册。用于每日开盘前的快速回顾和盘中决策参考。

---

## 一、核心心法

### 看山是山

> 今天盘面什么涨得最好，就是最强的方向。简单、直接。

- **去弱留强**：永远拿着市场最强的标的，龙头不可能逃出你的手掌心
- **不做预测，只做策略**：盘前写好多种剧本，开盘只是执行
- **三不心法**：
  - **不动心**：上帝视角，不被消息/股评/经验绑架
  - **不易心**：只做模式内交易，别人赚多少是别人的事
  - **心如止水**：毫无感情的交易机器

### 一根准绳

> **「假设你现在空仓，你还会买回这个标的吗？」**

- 会 → 拿着
- 有更想要的 → 换股
- 想空仓 → 卖出/减仓

## 二、每日流程

### 盘前准备（前一天晚上）

1. 复盘当天操作得失
2. 判断当前情绪周期阶段
3. 写好次日多种剧本：超预期怎么办 / 符合预期怎么办 / 不及预期怎么办
4. 列出持仓每只票的预期和应对策略

### 竞价六步法（9:20 起床，5 分钟完成）

| 步骤  | 看什么      | 判断要点                   |
| :-: | -------- | ---------------------- |
|  ①  | **持仓**   | 高于/低于/符合预期？决定处理策略      |
|  ②  | **龙头**   | 龙头溢价如何？判断情绪风向          |
|  ③  | **黑暗标的** | 昨日跌停/大面票有没有修复？亏钱效应是否减弱 |
|  ④  | **连板溢价** | 红多还是绿多？直观反映接力情绪        |
|  ⑤  | **一字封单** | 有没有十几亿/几十亿封单？判断方向强度    |
|  ⑥  | **批量概念** | 一字板是否集中同一方向？今天炒什么一目了然  |

### 盘中观察要点

- **9:30-10:00**：不急于动手，观察竞价信号是否被验证
- **10:00 前**：确认今日最强方向和最弱方向
- **尾盘 14:30-15:00**：关注涨速榜，捕捉资金回流方向

### 每日复盘框架

1. 今日操作得失（每笔交易的进出理由）
2. 情绪周期阶段更新
3. 龙头的状态变化（加速/分歧/盘顶）
4. 赚钱效应和亏钱效应的变化
5. 次日策略和剧本

## 三、市场节奏判断

### 情绪周期四阶段

```
冰点 → 复苏 → 高潮 → 退潮 → 冰点 → …
```

| 阶段 | 盘面信号 | 操作策略 |
|------|----------|----------|
| **冰点** | 高度板压制（3板以下）、涨停<30只、炸板率>40% | 空仓或极小仓试错 |
| **复苏** | 开山龙出现打破高度、赚钱效应回暖 | 积极试错开山龙 |
| **高潮** | 批量涨停（>80只）、跟风补涨全面扩散、边缘老龙涨停 | 重仓龙头或切补涨 |
| **退潮** | 高位负反馈、打板亏钱、中位先崩再传导龙头 | 减仓防守，不立危墙；退潮充分后有超跌反弹窗口（左侧分批，≤30%） |

### 关键判断信号

| 信号 | 含义 |
|------|------|
| 连板高度被压制到 3 板以下 | 冰点极限，大龙头孕育中 |
| 边缘化老龙头突然涨停 | **情绪溢出**——高潮即将转分歧，次日不接力 |
| 中位跟风先出现亏钱效应 | 分歧即将传导至龙头 |
| 炸板率 > 50% | 退潮确认或临近 |
| 恐慌极致 + 某方向逆势抗跌 | 冰点+新方向萌芽信号 |

### 大周期嵌套原则

> **小级别服从大级别。大周期正向时，小周期分歧是机会；大周期反向时，小周期高潮是炸弹。**

### 主要矛盾识别公式

```
阶段定位 → 核心变量 → 数据验证 → 动态切换
```

四个层级自上而下：全市场主要矛盾 → 市场风格主要矛盾 → 行业板块主要矛盾 → 个股主要矛盾

## 四、选股框架

### 龙头的五个「带头」

| 维度 | 标准 |
|------|------|
| **涨幅带头** | 一段主升行情以来的累积涨幅最高 |
| **启动带头** | 领先指数和板块，在市场最恐慌时打破局面 |
| **路径带头** | 开辟新的炒作路径，后续品种模仿跟随 |
| **赚钱效应带头** | 打出标杆后带动整个方向扩散 |
| **逻辑带头** | 有最正、最强的逻辑支撑 |

### 龙头确认三步走

| 名牌点 | 触发条件 | 确认程度 | 操作 |
|--------|----------|:---:|------|
| 第一次名牌 | N板确认主线（如4板确认华为线） | 30% | 分歧时可买入 |
| 第二次名牌 | 涨幅超越所有竞争者 | 60% | 分歧时加仓 |
| 第三次名牌 | 竞争者明确失败 | 100% | 龙头选手不计成本上车 |

> **每次名牌之后的分歧，都是买入窗口。级别越大的名牌，分歧买入越安全。**

### 龙头 ≠ 连板高标

- 龙头看累积涨幅，不看板数
- 龙头需要板块配合（小弟协同作战）
- 龙头在极端情绪中诞生（否极泰来处）
- 龙头必然打破现有市场运行状态（破局者）

### 板块内角色定位

| 角色 | 特征 | 参与价值 |
|------|------|:---:|
| **龙头** | 竞争胜出的领涨品种 | ⭐⭐⭐⭐⭐ |
| **中军** | 容纳大资金、逻辑最正、成交最大 | ⭐⭐⭐⭐ |
| **换手龙** | 前排一字不给机会时的替代 | ⭐⭐⭐ |
| **卡位** | 当日盘口最强的品种 | ⭐⭐⭐ |
| **补涨** | 龙头打出高度后的低位急速拉升 | ⭐⭐ |
| **跟风** | 龙头涨停后的被动跟随 | ⭐ |

> **永远不做跟风。龙头没有走弱，就不要做补涨。**

### 共振出妖股

三层共振（全部满足 = 最优交易窗口）：
1. **天时**：政策/宏观有利
2. **地利**：市场环境配合（指数止跌/回暖）
3. **人和**：合力形成（多类型资金同步做多）

## 五、进出场执行

### 买卖点速查（按标的身份）

| 身份 | 买点 | 卖点 |
|------|------|------|
| **龙头** | 每次分歧转一致都是买点 | 断板减仓 → 盘顶做T → 补涨共振下杀清仓 |
| **卡位** | 龙头分歧日，低位最先秒板 | 放量封不住就卖 |
| **补涨（缩量型）** | 龙头涨不动时的低位一字/连板 | 第一次开板放量即卖 |
| **补涨（换手型）** | 类似缩小版龙头，格局持有 | 断板时视主线强度决定 |
| **跟风** | 不买 | 赚钱就走，封板无力也走 |
| **套利标的** | 极致性价比位置低吸 | 有溢价就跑，拔屌无情 |

### 分歧与一致：核心买卖节奏

```
买在分歧 → 卖在一致
放量 = 分歧（买点）    缩量 = 一致（卖点）
```

**分歧转一致次数规律**：
- 第 1 次：大胆干，成功率高
- 第 2 次：谨慎，降低仓位
- 第 3 次：基本转强不了，不做

**分歧的传导路径**：
```
中位先崩 → 传导至高位龙头 → 龙头分歧转一致？→ 继续向上
                                    → 转不了一致？→ 瓦解崩溃
```

**板块 vs 个股强弱口诀**：
- 板块强、个股弱 → 个股真的弱
- 板块弱、个股强 → 个股真的强
- 板块强、个股强 → 你好我好大家好
- 板块弱、个股弱 → 一起完蛋

### 高潮次日铁律

> **高潮次日不接力。** 场外想买的都买了，没人接力只有断板。

高潮次日操作清单：
- [ ] 竞价反馈差 → 非核心第一笔抢跑
- [ ] 不接任何缩量秒板
- [ ] 核心股减仓，非核心清仓
- [ ] 等待分歧后再判断

### 弱转强信号

**弱转强是一个综合的局，不是看到分时拉就干。**

四个层次（自上而下验证）：
1. 大盘弱转强（指数共振的领涨品种才有确定性）
2. 板块弱转强（分歧转一致）
3. 同概念个股弱转强（卡位，后排卡前排要重视）
4. 个股自身弱转强（分时/隔夜）

**高成功率场景**：
- ✅ 指数经历过冰点恐慌回暖
- ✅ 主线题材刚经历第一次分歧，有转一致预期
- ✅ 龙头前期烂板，次日低位有强势一字助攻
- ✅ 题材分歧日，有标的主动引领资金回流

**危险信号（伪弱转强）**：
- ❌ 退潮期的弱转强（多是骗炮）
- ❌ 爆量弱转强（量爆太多 = 多空再次不一致）
- ❌ 高潮次日板块整体分歧中的个股弱转强
- ❌ 无量能萎缩+跌幅趋缓+领跌企稳等前置信号

### 三手法选择

| 手法 | 适用场景 | 风险 | 作者偏好 |
|------|----------|:---:|:---:|
| **打板** | 追求确定性，确认封板后买入 | 中 | ⭐⭐⭐ |
| **低吸** | 买在分歧，追求极致性价比 | 中 | ⭐⭐ |
| **半路** | 盘中确认方向后追入 | 高 | ⭐ |

> **手法只是工具，不是战法。** 必须配合情绪周期使用。冰点用打板确认，主升可低吸加仓，退潮不做。

## 六、仓位与风控

### 仓位-情绪周期对照表

| 情绪阶段 | 仓位建议 | 操作要点 |
|----------|:---:|------|
| 冰点 | ≤ 10% | 空仓或极小仓试错 |
| 复苏 | 30% → 50% | 逐步加仓开山龙 |
| 高潮 | 50% → 70% | 重仓龙头或切补涨 |
| 退潮 | ≤ 30% | 大幅减仓防守 |

### 退潮反弹中的两种策略

> 退潮不是"一跌不回头"，中间有超跌反弹窗口。详见 [[仓位管理#退潮中的仓位节奏：左侧分批 vs 右侧确认（2026.06.02 新增）|仓位管理]] 和 [[情绪周期#退潮反弹实战：左侧分批的节奏把握（2026.06.02 新增）|情绪周期]]。

| 策略 | 做法 | 适用场景 |
|------|------|------|
| **左侧分批** | 跌到目标位开始分仓低吸，越跌越买 | 有明确的估值/技术支撑位，提前看好方向 |
| **右侧确认** | 等放量阳线+趋势确认后进场 | 不确定性高时，宁可晚一步确认再买 |

> 两种策略的前提相同：**退潮中总仓位控制在 30% 以内**。区别在于进场的时机选择。

### 334 滚动仓位法

将账户分 10 份：**3 成老仓 + 3 成新仓 + 4 成备用**

| 场景 | 规则 |
|------|------|
| **老仓强势 + 新仓封死/浮盈** | 可动用备用金，盘中可满仓，收盘前减出次日备用 |
| **老仓强势 + 新仓炸板/走低** | 不再开新仓，快速处理老仓，备用金坚决不动 |
| **老仓弱势** | 优先处理老仓。新仓最多砍半，绝不急于扳回亏损而重仓 |

> **永不在单票上满仓。**

### 仓位-成功率公式

> **日内仓位不应超过你的交易成功率。**

- 成功率 30% → 日内持仓 ≤ 30%
- 成功率 40%+（作者水平）→ **日常**也就 3-4 成仓
- 只有主线明确、龙头确认时才把仓位加上去
- **高手满仓的条件极高，一年不到 10 次。韭菜天天重仓。**

> ⚠️ **两层规则的关系**：成功率公式是**日常安全基线**（大部分时间遵守），周期对照表是**阶段战术覆盖**（高潮/冰点等确定性阶段可突破基线）。两者不矛盾——高潮时因为市场容错率高，实际成功率远超平时的 40%，所以允许 50-70% 仓位；而平时混沌/震荡期，成功率基线就是仓位天花板。

### 铁律

- 🚫 绝不融资融券上杠杆
- 🚫 绝不在没有确定性的机会上重仓
- 🚫 绝不跟周期对着干
- 🚫 退潮期不立危墙，老手死于抄底
- ✅ 90% 的时间猥琐，10% 的时间重仓猛干

## 七、关键规则速查

### 买入清单

- [ ] 当前情绪周期阶段允许开仓吗？
- [ ] 这个标的是龙头/核心吗？（不是跟风）
- [ ] 有板块配合吗？（独立走势难成大妖）
- [ ] 是在分歧中买吗？（不追一致高潮）
- [ ] 想清楚失败了怎么办吗？

### 卖出清单

- [ ] 买入理由还在吗？（不在就卖）
- [ ] 标的是否走弱了？（走弱就卖，不等回本）
- [ ] 是高潮次日吗？（非核心先出）
- [ ] 关键支撑破位了吗？（破位先走，不赌）
- [ ] 空仓还会买回吗？（不会就卖）

### 十大铁律

1. **高潮次日不接力**
2. **退潮期不立危墙**
3. **永远不做跟风**
4. **买在分歧，卖在一致**
5. **第一次分歧转一致大胆干，第三次不做**
6. **仓位不超过成功率**
7. **弱转强是综合的局，不是单一信号**
8. **手法只是工具，必须配合周期**
9. **套利不格局，有赚就跑**
10. **看山是山，去弱留强**
