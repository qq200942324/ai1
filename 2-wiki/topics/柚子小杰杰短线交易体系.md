## 三层结构

### 道（哲学层）

- [[看山是山]]：三重境界，去弱留强，不执著于相
- 市场永远在变，唯一不变的是变化本身
- 「凡所有相皆为虚妄」——不要迷信任何固定模型

### 法（系统层）

| 概念 | 一句话 |
|------|--------|
| [[情绪周期]] | 冰点→复苏→高潮→退潮，顺应周期 |
| [[情绪与合力]] | 大家觉得好才是真的好，人气所向牛股所在 |
| [[龙头认知]] | 涨幅带头 + 启动带头 + 路径带头 + 赚钱效应带头 + 逻辑带头 |
| [[赚钱效应]] | 发现可模仿的盈利模式，跟随进场 |
| [[交易系统]] | 判断→进场→节奏→复盘，四大闭环 |
| [[预期与预期差]] | 金融资产价格取决于未来预期，赚预期差的钱 |
| [[共振与跷跷板]] | 共振出妖股，跷跷板需果断切换 |
| [[切换与二波]] | 节奏决定收益，二波需要多条件共振 |
| [[仓位管理]] | 随情绪阶段动态调整，小资金聚焦 |

### 术（技法层）

| 技法 | 用途 |
|------|------|
| [[买卖点]] | 可模式化的买卖信号，分时+K线双维度 |
| [[分时做T]] | 利用日内波动赚差价，辅助持股 |
| [[弱转强]] | 经典转折信号，分歧转一致的实战应用 |
| [[筹码分析]] | 筹码结构决定拉升阻力和效率 |
| [[打板低吸半路]] | 三种进场手法，打板确定性最高 |
| [[分歧与一致]] | 分歧中买，一致中卖 |
