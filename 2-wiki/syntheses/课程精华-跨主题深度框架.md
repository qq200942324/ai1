# 课程精华：跨主题深度框架

> 从「柚子小杰杰」完整课程体系中提炼的跨主题框架。这些内容不单独属于某一个概念，而是连接多个概念的方法论。

---

## 一、题材判断框架

### 三级穿透力判断

| 维度 | 大题材 | 小题材 |
|------|--------|--------|
| **消息源头权重** | 国家级政策 > 产业变革 > 地方政策 | 行业自律、单个公司事件 |
| **影响覆盖广度** | 长产业链、多分支、高容量（能容纳机构资金） | 单一细分方向 |
| **落地细则** | 有量化指标（数字）、有时间节点 | 空泛表态（「大力发展」） |

### 持续性三要素

| 要素 | 关键判断 |
|------|---------|
| **催化剂密度** | 连续落地 > 一次性释放；政策+产业双轮驱动最强 |
| **资金介入深度** | 机构+游资+散户三方参与 > 纯游资炒作；机构调研+加仓=延寿 |
| **参与性** | 有换手、能进出 > 批量一字板（一波流，走不远） |

### 替代成本——最关键的判断

> **定义**：一个新题材要成为市场唯一主线，需要花多大力气打败其他竞争者。

| 情境 | 替代成本 | 结论 |
|------|:---:|------|
| 前主线退完 + 情绪/指数双弱 + 空窗期 | **零** | 不需要打败谁，站出来就是主线（如 DC、华为周期） |
| 多方向轮动 + 强指数（如雅江） | **极高** | 即使拼赢也是惨胜，动能耗尽 |

> **「轮动当中角逐出最后题材成为主线的可能性几乎为零。」**

## 二、接力的本质：传接

> 新时代接力不再是简单的连板晋级，而是资金共识的传递。

### 市场变了什么

| 过去 | 现在 |
|------|------|
| 纯粹情绪博弈（看强是强） | 政策解读 + 产业洞察 + 量化对抗 + 合规控制 |
| 龙头战法/接力战法一招通吃 | 多维竞争，单一模式选手被淘汰 |
| 连板高标 = 情绪风向标 | 连板高标高度不再代表市场客观情绪 |

### 连板高标的抱团效应

- 当题材不适合大规模持续炒作 → 做多动能只能聚焦 → 出现连板高标+半生次高的抱团
- 例：西游+路基、程飞+利君
- 这种高标**不带动连板风格回归**，也不带动整个题材

### 断板后的新常态

> **断板 ≠ A杀。断板后整理再上已经是常态。**

- 断板后承接住 → 构筑平台 → 可能走出短线趋势甚至新高
- 实战方法：每天把断板3天内的标的汇总成分组观察，跌破平台就删
- 弱转强的条件变了：必须有**唯一性**（真金白银竞价反核 + 区别于其他标的）+ **题材热度还在**

### 低位传接仍然有效

- 高标断板 → 切低位首板/二板（传接），但不能切中位
- 中位股是亏钱效应最先蔓延的地方

## 三、周期全解：多方循环

### 核心哲学

> **「凡所有相，皆为虚妄。」**——周期的阶段划分只是表象，是人为贴的标签。

- 所有可量化的理论（技术指标、龙头战法、周期阶段划分）都有**阶段性和片面性**
- 市场唯一不变的真理：**市场一直在变**
- 游资的核心价值：**最快跟随市场、最快适应变化**

### 多方循环（可操作期）

```
酝酿 → 部分盈利 → 消息发酵 → 扩散（增量进场）
→ 主升（不断验证+加仓） → 群体效应强化
→ 高潮（最后资金涌入、完全丧失风险意识）
→ 犹豫（盘顶）→ 做多动能衰减但不会立即死
→ 期待逐渐消亡 → 退潮完成 → 新循环开始
```

### 三维共振

> 周期是**指数 × 题材 × 龙头**三者动态互动的结果，没有简单的线性关系。

- 指数强 → 增强题材和龙头的容错率
- 题材强 → 带动市场热度和指数
- 龙头强 → 维持场子热度，帮助分歧转一致

### 周期不变与变

- **不变**：启动总会结束，结束之后新一轮总会开始（闭环）
- **变**：中间各个环节、各种演变的多样性——每个周期都不一样
- **结论**：无法通过阶段划分去量化，要靠每天的复盘+情绪观察

## 四、二波的条件

> 二波是赚钱效应延伸的路径，但条件极其苛刻。

### 四类二波场景

| 场景 | 触发条件 | 实例 |
|------|---------|------|
| 情绪爆炸推动 | 龙头盘顶时低位热情如火，做多动能溢出 | 九安医疗 |
| 补涨/切换失败 | 低位强度不足，切换不出新题材 | 龙头自己带队走二波 |
| 大周期踏空 | 有高度但散户参与性差，踏空者巨多 | 大市值趋势二波 |
| 周期龙主导 | 大周期背景 + 高度共识 | 浙江建投、翠微 |

### 筹码条件

- 断板后**量能不能明显缩减**——需要充分换手
- 筹码从低位**批量转移到高位聚集**——统一起跑线
- 高位平台突破时**不一定要放量**（因为换手已经充分）

### 二波不死条件

- 题材远远没炒完、分支没挖完
- 市场上没有能构成威胁的替代方向
- 龙头确认后，补涨和跟风都只是备胎

## 五、打板/低吸/半路的选择框架

> 不是喜欢什么就用什么，而是**在恰当的时候用恰当的手法**。

### 打板的四种目的

| 目的 | 场景 |
|------|------|
| **抢筹码** | 转世板、高位分歧做补涨、政策面解读、强行纠偏 |
| **追求确定性** | 对抗胜利的上板确认、连板晋级的确定性在板上 |
| **追求溢价** | 顺势而为的大市值首板、四大历程封板 |
| **情绪确定性** | 批量打板（打的不是个股，是情绪回暖的确定性） |

### 低吸的应用

- 趋势股回踩均线（五日线/平台下轨）
- 龙头分歧日分歧延续后的低吸
- 行情没走完 + 意外消息面恐慌杀跌 = 绝佳低吸点

### 半路的应用

- 适用于趋势加速的初期
- 需要叠加逻辑判断（不是纯技术追涨）

## 六、题材挖掘方法大全（PPT 深化）

> 从 PPT 课件提炼的五种题材挖掘路径 + 信息获取渠道。

### 五种挖掘路径

| 路径 | 方法 | 实例 |
|------|------|------|
| **政策相关** | 关注中央媒体/新闻联播；政策市特征：顶层提出+多部门协同+时间窗口合适 → 大概率大题材 | 中特估、新质生产力、重组 |
| **大局观推演** | 精准捕捉国内经济关键矛盾点，提前推演猜想 → 市场走出来时更快把握 | 化债→中特估；封城复苏→新能源；贸易战→白酒抱团/一带一路 |
| **主线分支** | 主线明确后，沿产业链上下游挖掘；逻辑纯正不一定涨最多但大概率涨 | AI：AIGC/光模块（优势方向）先涨 |
| **突发事件** | 依赖长期经验积累的观察力+反应力；优先选最快启动/资金最关注标的 | 核辐射→大湖股份；信息泄露→国华网安 |
| **涨价逻辑** | 关注核心产品涨价幅度/原因/持续性/预期高度；大宗商品看期货 | 航运提价→中远海控；碳酸锂涨价 |

### 信息获取渠道

| 类型 | 渠道 |
|------|------|
| 政策权威 | 新华网、人民日报、央广网 |
| 行业数据 | Wind EDB、生意社、百川、卓创 |
| 资讯快讯 | 财联社 24h 快讯、选股宝 |
| 研报 | 财联社、脉搏汇金、知丘 |
| 论坛 | 雪球、淘股吧、韭菜公社、开盘啦 |
| 舆情 | 抖音/公众号搜索"消息面总结/汇总"，社交媒体热门话题 |

### 盘中快速判断

- 大机会不用抢，要抢的不会是大机会
- 盘中追消息涨的多数是坑
- 盘中异动 → 快速判断是否有预期差 → 是否符合题材判断框架

## 七、三种风格轮动策略（PPT 深化）

> 短线连板 × 趋势的实战平衡框架。

| 当前状态 | 策略 |
|---------|------|
| **强趋势 + 强情绪** | 保持趋势底仓 + 连板加仓猛干，等待转势离场 |
| **强趋势 + 弱情绪** | 等趋势加速后低吸趋势筹码，或做低位连板试错 |
| **弱趋势 + 强情绪** | 保持情绪品种仓位，右侧亏损离场不加仓；观察趋势承接/短线补涨 |
| **弱趋势 + 弱情绪** | 全面规避连板（通杀风险）；等待拐点，参与共振风格 |

> 水平不够的：选一个风格躺好——要么大趋势，要么等短线回流再做。
