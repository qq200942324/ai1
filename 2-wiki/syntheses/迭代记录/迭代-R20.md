## 8 个新发现详解

### 1. 情绪溢出信号 ⭐（→ 情绪与合力 + 情绪周期）

> 来源：[[1-sources/[2025-06-17]论情绪合力分歧与一致及趋势判断]]

**原文**："如果发现有边缘化的老龙头出现涨停，大概率是要高潮结束或者要分歧的表现。踏空的人连他妈后排都抢不到了，去抢场内其他的人气标的。"

**框架**：老龙头涨停 = 踏空资金的最后出口 = 主流已无买盘空间 = 高潮末端预警信号。

### 2. 轮动市五条生存法则 ⭐（→ 交易系统）

> 来源：[[1-sources/[2025-07-02]行情揭秘]]

五条法则层层递进：
1. 轮动题材分歧转一致的表现形式会不断弱化
2. 接力思维毒如砒霜，绝不奢求强更强
3. 无从下手→往低切；低切也挨揍→往抱团切
4. 抱团也不好→消息面刺激的无序短打（难度极高）
5. **开始不好做→立马控仓，死守前期利润**

### 3. "多空双杀"概念 ⭐（→ 情绪周期）

> 来源：[[1-sources/[2025-07-06]多空双杀下周策略]]

**原文**："多头追高死了，空头早上割肉也死了。"

放量冲高回落 → 追涨和割肉两边同时亏钱。这是一个可复用的行情命名，便于后续快速识别同类模式。后续三种路径概率排序：震荡 > 暴跌 > 爆拉。

### 4. 趋势低吸级别升级 ⭐（→ 买卖点）

> 来源：[[1-sources/[2025-07-06]多空双杀下周策略]]

**原文**："只要他们走成了周线级别趋势行情，那么低吸就要从日内的分时，到日线级别，到现在就需要等周线级别的分歧再去考虑低吸了。"

| 趋势级别 | 低吸等待级别 |
|----------|------------|
| 日内 | 分时低点 |
| 日线趋势 | 日线级别回调 |
| 周线趋势 | 周线级别分歧 |

### 5. 二波十律完整框架 ⭐⭐（→ 切换与二波）

> 来源：[[1-sources/[2025-07-11]二波二波二波的秘密]]

| 维度 | 规律 |
|------|------|
| 周期地位 | 只有龙头/龙二才有二波资格 |
| 三种触发 | ①主升火爆→低位热情→龙头二波 ②低位强度不足→龙头被重新聚焦 ③散户踏空多→再次上车需求 |
| 筹码条件 | 短期稳定→高位聚集→具备爆发力 |
| 风格区分 | 短线/断板风：需高位平台突破涨停；趋势风：势大力沉分时创新高不需涨停 |
| 被动二波 | 监管不允许一波→迂回战略 |
| 本质定义 | 二波=赚钱效应延伸的路径，具有被模仿性 |
| 真伪区分 | 主动势大力沉上板 vs 被动低位拿命顶 |

### 6. 流动性高点出场法 ⭐（→ 买卖点）

> 来源：[[1-sources/[2025-07-12]宝再见了我很爱你]]

**四维判断**：人气还在 + 一致预期 + 低位死顶 + 8%以上巨量换手仍封不住 = 所有潜在买盘已耗尽 = 最佳出场点。

### 7. 分歧充分度判断（老龙头反弹法）⭐（→ 分歧与一致）

> 来源：[[1-sources/[2025-08-08]啊啊啊分歧加大惨遭炸板巨面]]

**原文**："其实看到雅下反弹我挺开心的，说明市场是真的是一个比较充分的分歧。"

**框架**：分歧日看到前期老龙头（情绪连板属性）反弹 = 资金已轮完所有方向 = 分歧足够充分 = 后续转一致的概率上升。

### 8. 牛熊分化的分歧定性 ⭐（→ 分歧与一致 + 情绪周期）

> 来源：[[1-sources/[2025-08-14]今日分歧继续撸]]

**原文**："今天的分歧放在熊市里面，会是一个极冰。但是现在是流动性充裕的牛牛，表现出的只是分化分歧，而不是全面的分歧。"

| 环境 | 同级别下跌的含义 |
|------|----------------|
| 熊市 | 极冰 → 系统性风险 → 空仓 |
| 牛市/充裕流动性 | 分化分歧 → 正常调整 → 继续撸 |

## 其他重要框架发现（🔶 部分中提炼）

### 趋势风格三吊打定律

> 来源：[[1-sources/[2025-08-05]左长城右上纬继续干强趋势]]

"短线趋势吊打短线连板，连续换手吊打连板加速，弱转强吊打强更强。"

### 赚钱效应传接路径

> 来源：[[1-sources/[2025-08-12]10亿以下不用分仓]]

"理解市场高流动性以及高风偏的底层逻辑，再找到赚钱效应的传接路径。"

### 趋势三要素

> 来源：[[1-sources/[2025-08-15]天佑我大中华]]

"围绕着人气高度容量，趋势手法反复干就很好了。" → **人气（辨识度）+ 高度（空间）+ 容量（流动性）** = 趋势标的筛选铁三角。

### 三维环境确认

> 来源：[[1-sources/[2025-08-07]感谢市场赚钱效应很好]]

"指数环境不错，流动性不错，情绪不错，赚钱效应很好。" → 三个维度同时向好 = 高容错率环境。

### 完整周策略推演闭环

> 来源：[[1-sources/[2025-07-26]吃肉但是费劲儿的一周]]

周一抢筹铺仓→周二加仓→周三踩坑了结+自省→周四完美执行→周五验证不切换。关键：**应对优于预测**，踩坑后不自责而制定应对策略。

### "事不过三"节奏定律

> 来源：[[1-sources/[2025-06-17]论情绪合力分歧与一致及趋势判断]]

小主升第1天买→第2天加速（想买的都在第2天买）→第3天惯性高开高走=阶段性离场时机。
