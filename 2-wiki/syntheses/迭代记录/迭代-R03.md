## 三、概念页深化详情

### 3.1 [[交易系统]] — 新增"核心心法层" + "竞价六步法"

**来源**：`2.核心心法07.08.md` + `2.周复盘+交易系统-1014.md`

**新增内容**：
1. **不动心 & 不易心**：超短龙头选手的两种核心心态
   - 不动心：以完全客观的上帝视角看待市场，不被消息面左右，不被经验主义绑架
   - 不易心：专注自己的交易系统，模式≠手法，模式=环境+周期+窗口+底层逻辑的可重复体系
2. **竞价六步法**：每日 9:25 开始的 5 分钟固定流程
   - ①看手上持仓 ②看龙头股 ③看亏钱效应标的 ④看连板溢价 ⑤看一字板封单 ⑥综合判断修改策略
3. **仓位递增实操**：起步 15% 试错 → 主线确认 38% → 顺势加仓 65% → 高潮 89% → 减仓
4. **中位监控原则**：中位掉队品种的负反馈程度判断主线健康度

### 3.2 [[龙头认知]] — 新增"发育节奏" + "五顺原则" + "强制发生关系"

**来源**：`3.龙头发育与盘顶及其对应的跟风补涨节奏3.md` + `4.真正的龙头系统——一套交易四通.md`

**新增内容**：
1. **三次名牌节点**（以捷荣/华为周期为例）
   - 第一次名牌：主线确认（四板）
   - 第二次名牌：涨幅超越竞争者（60%-80%确认）
   - 第三次名牌：反包二波（100%确认全市场总龙）
   - 每次名牌后的第一次分歧 = 重仓买入点
2. **五顺原则**：顺市场环境、顺周期环境、顺指数、顺风格、顺趋势
3. **强制发生关系**：对于大周期龙头，最优解是先把筹码拿到手，不等最优介入点
4. **盘顶多因素**：周期深度、题材故事量、踏空盘、分支横向挖掘、跨年预期共同决定盘顶时长
5. **监管扭曲周期的应对**：连续监管打压后资金不再锚定周期龙做二波，转为趋势蛇皮走位品种

### 3.3 [[主要矛盾与核心论]] — 新增"宏观落地框架"

**来源**：`2024宏观.md` + `2024年中宏观展望.md` + `2024年年中宏观演讲.md` + `3.md` + `20251223周二唠嗑.md`

**新增内容**：
1. **五个口袋理论**：财富不会消失只会转移——现金/股权/债权/实物/外汇五口袋
2. **化债三路径**：创造新增量（出海）、温和通胀（CPI目标3%）、软着陆分批爆雷
3. **四象限投资框架**：风险偏好（高/低）× 品种类型（大型价值/小型题材）
4. **A股转型逻辑**：从"融资市"到"分红市"——垄断+高分红+强现金流=未来长牛方向
5. **牛市四大分类**：红利银行/左手资源右手科技/顺周期/阶段主题
6. **科技子赛道逻辑排序法**：环比利润率提升+环比增速提高为判断标准

### 3.4 [[赚钱效应]] — 新增"核心公式" + "梯队选择" + "轮回模型"

**来源**：`3.周复盘+捕抓赚钱效应-10.21.md` + `3.发现捕获赚钱效应以及龙头交易节奏（竞业达为例）.md`

**新增内容**：
1. **捕获赚钱效应核心公式**：通过自身超预期打出识别度，引发全新的、可持续的、可供模仿的赚钱效应标的
2. **赚钱效应所在梯队判断**：晋级率最高的梯队 + 超预期最集中的梯队 = 赚钱效应所在
3. **完整轮回模型**：追高赚 → 追高低吸都赚（主升）→ 低吸赚追高亏（高位分歧）→ 都亏（退潮）→ 低吸赚（孕育新周期）
4. **死磕一条线原则**：双主线行情容易两边都赚不到，选择先带领盘面与情绪共振扭转亏钱效应的那条线死磕到底

### 3.5 [[打板低吸半路]] — 新增"缩量一封规则" + "炸板区分"

**来源**：`3.技法1.0 07.19.md`

**新增内容**：
1. **三种不能打的缩量一封**：分歧日的缩量一封 / 无加速预期的缩量一封 / 龙头负反馈时跟风的缩量一封
2. **被动炸板 vs 主动炸板**：情绪分歧导致→被动炸板（可打回封）；别人都没事只有你炸→主动炸板（真弱）
3. **理解力优于手速**：打板核心不是装备和手速，而是复盘时就预期它能涨停
4. **容错率指标**：当天买错次日是否有修复？核按钮是否有溢价？容错率高→积极进场，低→收缩

### 3.6 [[预期与预期差]] — 新增"三层逻辑"

**来源**：`4.详解预期与预期差.md`

**新增内容**：
1. **预期差三层逻辑**：价值逻辑（买在跌破价值过程中）、估值逻辑（买在股价未兑现估值过程中）、情绪逻辑（买在情绪上升过程中，预期有对手盘更高价接货）
2. **弱转强三维度判断**：纵向对比（高度和地位）、横向对比（同梯队竞争对手）、细节把控（周期背景+盘口状态）
