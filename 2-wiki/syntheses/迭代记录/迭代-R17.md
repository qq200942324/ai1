## 二、11 大方法论新发现（按重要程度排序）

### ⭐⭐⭐ 1. 脱钩边际递减法则（4/8 加仓打满）

> "34%和50%有差吗...基本就已经停滞了...贸易已经停滞了，再加50%其实停滞和贸易脱钩是没差的，只能影响开盘价。"

当关税加到经济脱钩临界点时，进一步加税对资产定价的边际影响趋零。这不是"利空出尽是利好"，而是从经济学原理推导的：当双边贸易已实质停滞，再加关税只影响心理面（开盘价），不影响反制方向的做多逻辑。

→ 反哺：[[主要矛盾与核心论]]

### ⭐⭐⭐ 2. 底部确认10维观测清单（3/31 指数情绪双杀）

```
①割肉盘出现（好事） ②银行红利护盘是否被反杀 ③资金回流方向
④各指数下影线长度对比 ⑤小票vs大票探底回升程度 ⑥H股是否同步
⑦上证50上下引线比例 ⑧AI50振幅和放量对比 ⑨涨跌停比/连板高度
⑩群众情绪（评论区骂声=接近底部）
```

将底部确认从模糊的"情绪冰点"升级为10个可观测的维度。3/31当天的拉回5个都不满足→判断只是"有可能进入横盘小区间"而非真见底。

→ 反哺：[[情绪周期]]

### ⭐⭐⭐ 3. 四种路径推演框架（4/16 又涨停了全都是运气啊）

将周期后续走向归纳为四种可量化路径：
- **路径A**：再次放量+分歧转一致 → 走出规模短线行情
- **路径B**：量能不变+人气品种趋势新高 → 为周期续命
- **路径C**：延续轮动反抽一次比一次弱 → 集体GG
- **路径D**：宏观巨变 → 推倒重来等待第二只脚

每种路径对应不同的仓位和策略，从"猜方向"变成"准备四种剧本"。

→ 反哺：[[情绪周期]] + [[预期与预期差]]

### ⭐⭐ 4. 逻辑跌停 vs 情绪跌停二分法（4/8 加仓打满）

> "这些负反馈的票...不是短线炒作情绪退潮导致的批量跌停...是逻辑上的、是独立的，不会影响反制方向的做多。"

千股跌停时区分两类跌停：**逻辑跌停**（自身基本面/外部冲击/独立事件）不影响板块做多逻辑；**情绪跌停**（市场恐慌传导/短线退潮）需要避险。这个区分是极冰日敢于加仓的核心认知。

→ 反哺：[[情绪周期]]

### ⭐⭐ 5. 筹码向上聚合的爆量竞价方法论（4/14 村里有个姑娘叫小芳）

四种买入模式分类：
- **一字首开+竞价爆量** → 竞价直接上
- **筹码向上聚合+放量但不爆量** → 需等盘中换手确认
- **反弹筹码向上聚合+竞价爆量** → 竞价就上
- **中间辨识度首开** → 可不上或小仓

核心原则："不是筹码向上聚合+爆量就能直接上的，需结合题材是否还有预期来筛选。"

→ 反哺：[[买卖点]] + [[弱转强]]

### ⭐⭐ 6. 茅东招救市晴雨表（4/7 十万火急提前发文）

> "盘前A50期指连续和恒生期指连续并没有开盘下探后立即拉回，代表了今天救市的力度并不大。"

用茅台、东方财富、招商银行三只权重股的开盘表现判断国家队当日态度和救市力度。力度不大→不追开盘反弹→等盘中反杀低吸。

→ 反哺：[[主要矛盾与核心论]]

### ⭐ 7. 边界效应递减法则（4/5 华尔街恐慌）

> "如果这一次还是出现低开高走，然后抗住了，那么后续这个事件对龙哥影响的边界效应就会减小。"

同一宏观冲击反复发生时，市场定价效率提高，后续反应强度递减。这是行为金融学在A股的实战应用。

→ 反哺：[[主要矛盾与核心论]]

### ⭐ 8. 补涨必要性条件（4/10 炸裂不行了）

> "这个周期没有真正角逐出那个老大，虽然市场开始有低位补涨的苗头，但是没有必须做补涨的需求。"

不是所有行情都需要做补涨——只有当周期龙头明确+高度天花板清晰时，补涨才有逻辑支撑。"没有老大不必做补涨"是对常见做法的重要修正。

→ 反哺：[[龙头认知]]

### ⭐ 9. 周期级别降级框架（4/15 惊天巨物）

> "如果不能完成一个大级别的分歧转一致，那么大周期的条件就会因为这个发育过程低于预期而止步于小周期。"

周期不是固定的级别——它会因为发育过程的成败**动态降级或升级**。大分歧转一致成功→大周期；失败→小周期。

→ 反哺：[[情绪周期]]

### ⭐ 10. 极冰日操作SOP（4/7 十万火急提前发文）

五步完整流程：①判断救市力度（A50/恒生期指）→ ②等待盘中反杀（不追开盘反弹）→ ③翘板/低吸核心标 → ④建涨停观察池 → ⑤留活水等次日分化加仓。

→ 反哺：[[情绪周期]] + [[仓位管理]]

### ⭐ 11. 弱势抱团三段论 + 量能是外力框架

- **三段论**：业绩保障 + 不站边（独立逻辑）+ 有方向指引（先行者反包板）
- **量能是外力**："量能就是外力，就是打破原有运动状态的力量"——用物理学类比市场趋势

→ 反哺：[[龙头认知]] + [[赚钱效应]]

## 四、R17 特殊意义

本轮是**30轮迭代中方法论密度最高的轮次之一**。407关税冲击是2025年最重要的宏观事件，作者从3月下旬的龟缩防守→4/2关税落地→4/7千股跌停→4/8满仓打满→4/10收网→芳芳丽丽周期的完整演绎，覆盖了"宏观外部冲击→极冰→暴力反弹→分化抱团"的全链条。

**核心叙事弧**：
```
3.25 退潮防守 → 3.31 出现底部信号 → 4.02 地量拿货 → 4.07 千股跌停加仓 
→ 4.08 关税升级反而打满 → 4.09-10 持筹收割 → 4.14-18 芳芳丽丽抱团
```

**最核心的认知贡献**：脱钩边际递减——当关税加到实质脱钩后，边际影响趋零。这一认知让作者在4/8（关税再升级至104%）当天不仅不慌、反而加仓打满——成为本轮操作的最大胜负手。
