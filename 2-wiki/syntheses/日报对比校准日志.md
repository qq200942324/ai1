### 2026-05-27 · 震荡误判为退潮 + 仓位框架偏差 + 防御跷跷板误读

> 来源：[[2-wiki/data/daily_analysis]] vs [[2-wiki/summaries/杰哥复盘/2026-05-27 杰哥复盘 - 嘲讽与震荡应对]]

#### 逐维度对比

##### 1. 情绪阶段判断 ⚠️ 核心差异

| | Claude 日报 | 杰哥复盘 |
|------|:------:|:------:|
| **定性** | 退潮期中段 | 震荡修复 |
| **依据** | 7/8 指数下跌、科创50 连跌 4.2%、防御板块跷跷板 | 冲高回落是博弈性使然、科技内部仍有强势分支、不排除创新高 |
| **倾向** | 🔴 偏悲观 | 🟡 偏中性 |
| **策略** | 30% 仓位防守 | 底仓雷打不动 + 活仓高抛低吸 |

**差异分析**：我对「退潮期」的判断过于机械——只看到指数普跌和防御板块异动，忽略了两个关键事实：
- 杰哥指出的"很多科技股盘中给了冲高红盘减仓机会"→ 这不像是退潮期的"全面亏钱效应"
- 杰哥强调"短期震荡，中长期没走完"→ 大级别周期仍在向上的框架内

**教训**：退潮期的标准是"批量跌停 + 高位龙头全面负反馈"，而今日只是"科技内部高低切换 + 部分兑现"。我混淆了「震荡调整」和「退潮」——参考 [[情绪周期]] 中震荡阶段的特征。

##### 2. 仓位策略 ⚠️ 框架差异

| | Claude 日报 | 杰哥复盘 |
|------|:------:|:------:|
| **框架** | 单一比例（30%） | 底仓 + 活仓拆分 |
| **逻辑** | 退潮→降仓位 | 震荡→底仓不动 + 活仓做波动 |
| **灵活度** | 固定比例，僵化 | 根据能力圈动态调整 |

**差异分析**：杰哥的框架更精细——不是简单说"几成仓位"，而是：
- **底仓**：雷打不动，是中长期看多的核心仓位
- **活仓**：看得懂就做，看不懂就等，做高抛低吸

这与 [[仓位管理]] 中的「334 滚动法」精神一致，但在震荡市中做了适配——底仓 + 活仓的拆分为震荡市提供了比单纯降仓位更好的操作框架。

##### 3. 防御板块的判断 ⚠️ 策略差异

| | Claude 日报 | 杰哥复盘 |
|------|:------:|:------:|
| **态度** | 防御跷跷板=退潮信号，切换到防御 | 坚决不切低位红利 |
| **理由** | 避险资金流入防御 | "和过去两年一样，宁愿降仓位做科技内部轮动，也不切" |
| **实践** | 选股池无防御持仓 | 继续持有信W、利通T（科技线） |

**差异分析**：我把"防御板块异动"当成退潮的确认信号，但杰哥的经验是——防御板块的拉升更多是短期的跷跷板效应，不代表主线终结。他的做法是：不否认跷跷板的存在，但选择"降仓位+科技内部轮动"来应对，而非"切去防御"。

这在 [[共振与跷跷板]] 框架中有一个关键启示：跷跷板的出现不代表需要完全换方向，有时候只是"别人在跷跷板那头玩，我在主线这边等"。

##### 4. 盘面微观信号 🆕 日报缺失

杰哥复盘中有一个我完全没捕捉到的信号：

> **「创业板盘中超越上证指数，成了很多博弈者的兑现点。」**

这是典型的 A 股博弈心理——整数关口/心理锚点的突破被当作"利好出尽"的兑现时机。这是 [[分歧与一致]] 和 [[预期与预期差]] 框架下的经典案例。

**我日报的缺失**：只看到了"创业板独红 +0.07%"，没有进一步解读这个"独红"背后的博弈含义——盘中冲高超越上证后回落，说明是兑现而非真正的独立走强。

**差异分析**：我对风险的关注点偏宏观（AI 价格战、美伊），而杰哥更关注市场自身节奏（6 月时间窗口、提前兑现的博弈性）。

### 2026-05-28 · 退潮误判重犯 + 选股池方向单一 + 海外/国产链未区分 + 缩量误读 + 操作计划不匹配

> 来源：[[2-wiki/data/daily_analysis]] vs [[2-wiki/summaries/杰哥复盘/2026-05-28 杰哥复盘 - 易中天新高与三波龙头]]

#### 逐维度对比

##### 1. 情绪阶段判断 🔴 重犯错误

| | Claude 日报（初版） | Claude 日报（修正后） | 杰哥复盘 |
|------|:------:|:------:|:------:|
| **定性** | 退潮末段修复性反弹 | 震荡修复 | 震荡修复 + 散装主升大周期 |
| **依据** | 科创50 两日 -4.2% + 缩量反弹 | 无顶部结构 + 海外链新高 + 多方向共存 | 大票翻倍但未构造顶部结构、没破位就不看空 |
| **策略** | 50% 谨慎参与 | 50% 底仓+活仓+机动 | 底仓不动 + 活仓波动 + 多方向做多 |

**差异分析**：这是连续第二天把「震荡修复」误判为「退潮」。5/27 我判退潮中段、5/28 我判退潮末段——两天都在退潮框架里打转。杰哥的实际判断始终是「震荡修复 + 散装主升大周期」。核心差异在于退潮的确认标准——杰哥要求「顶部结构（双头+跌破颈线）」才算退潮，而我看到指数连跌就触发退潮判断。

**教训**：这是框架级系统性偏差——我的默认倾向是「跌了就往退潮靠」。需要在分析模板中增加一个强制检查项：**「退潮确认清单」**——在判定退潮前，必须先确认（1）核心标的是否破位（2）是否形成双头结构（3）颈线位是否跌破（4）龙头是否全面负反馈。清单未完全满足 → 不能判退潮。

##### 2. 核心矛盾 🔴 完全错误

| | Claude 日报（初版） | 杰哥复盘 |
|------|:------:|:------:|
| **定性** | AI 硬件强势反弹 vs **缩量无法确认反转** | **海外链持续加强** vs 国产链分歧调整 |
| **核心逻辑** | 缩量=危险信号 | 缩量=震荡修复的正常特征，不构成风险 |
| **两链区分** | ❌ 没有区分 | ✅ 海外链（CPO/PCB上游）先涨+国产链（设备/存储）跟随 |

**差异分析**：我完全没区分海外链和国产链的节奏差异。杰哥明确指出「海外链持续加强，易中天新高」「国产区调整但未破位」——两条链处于不同阶段。我把它们混在一起，导致：海外链的强势被我误读为「反弹」，国产链的分歧被我误读为「退潮」。

**教训**：未来的分析模板必须强制区分三条链——🔵海外链（CPO/光模块/PCB上游）、🟢国产链（半导体设备/存储/国产算力）、🟡情绪/短线。每条链判断独立，不能混。

##### 3. 选股池 🆕 连续两天方向单一

| | Claude 日报（初版） | 杰哥复盘 |
|------|:------:|:------:|
| **覆盖方向** | 100% AI 硬件（10/10） | 多方向：情绪（辽能三波）+ 算力（利T）+ PCB上游（宏H）+ 商航（信W）+ 短线（生意/泰永） |
| **选股逻辑** | screen_stocks.py 六维框架 → 几乎全筛出 AI 硬件 | 散装主升思维：每个方向找核心标的死缠烂打 |
| **操作风格** | 分散持有 10 只，每日轮换 | 聚焦 4-5 个核心做 T + 波段 + 短线各司其职 |

**差异分析**：5/27 的问题（选股池全 AI 硬件）在 5/28 不仅没改善，反而更严重——修正选股池时换了标的但依然全是 AI 硬件。杰哥的持仓展示了一个关键理念：散装主升大周期中，「多方向都有机会」，不需要把所有仓位压在同一个方向上。他的辽能三波、利T死缠翻倍、信W翻倍——分散在不同方向，各自独立赚钱。

**教训**：选股池的生成框架（screen_stocks.py 六维）可能天然偏向「当前最强的板块」→ 导致方向单一。需要在框架层面加入「方向多样性」约束，强制覆盖至少 3 个独立方向。

##### 4. 缩量的误读 🆕 新偏差类型

| | Claude 日报（初版） | 杰哥复盘 |
|------|:------:|:------:|
| **缩量=？** | 🔴 危险信号，反弹可能夭折 | 🟡 正常特征，不特别关注 |
| **逻辑** | 缩量=增量资金不足=反弹无持续性 | 缩量=抛压减轻（震荡修复中的正常现象） |
| **是否构成退潮证据** | 我：是 | 杰哥：否（他甚至没提缩量） |

**差异分析**：我把缩量当成退潮/反弹无持续性的证据，但杰哥完全没把缩量当作问题。原因在于：退潮期的缩量是「买方消失」，震荡修复期的缩量是「抛压减轻+资金聚焦」——同样的缩量在不同阶段有完全不同的含义。

**教训**：缩量不是独立的危险信号，必须结合阶段判断。在「退潮」中缩量=恶化，在「震荡修复」中缩量=正常。**不能用一个指标跨阶段套用**。

##### 5. 操作计划与选股池不匹配 🆕 流程错误

修正选股池时，我更新了第七节（选股池表）和第八节（持仓分配），但完全忘了同步更新第十节（个股操作计划）：
- 操作计划中仍有西安奕材-U、亨通光电、佰维存储、芯原股份——这 4 只已在选股池中移除
- 选股池新增的联特科技、中兴通讯、华天科技、英维克——操作计划中完全没有

**教训**：日报的三节（七·选股池 / 八·持仓分配 / 十·操作计划）是强耦合的，改一处必须同步改其他两处。以后修正日报时，改完选股池必须立即对照操作计划检查一致性。
