```

2. 科创50 跌破 1800 关键支撑（收 1751）
3. 上证50 独涨 = 防御跷跷板全面启动 = 资金大规模撤离科技
4. 20d Top30 中 16只下跌 + 8只大跌超5% = 赚钱效应快速收缩
```

```
趋势主升（5月初-5月中旬）
    ↓ 大票从战争坑翻倍
    ↓
```
