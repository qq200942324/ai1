# 散户

> 「又菜又爱玩。这里无关钱多钱少的问题——只要你水平足够菜，你就是散户。」——杰哥

---

## 定义与内部分层

散户不是一个资金量概念——是一个**认知水平**概念。杰哥特别强调：

> 「游资钱不一定多，但他一定牛逼。大户钱一定多，但他不一定牛逼。」

| 类型 | 资金体量 | 认知水平 | 行为模式 |
|------|------|:---:|------|
| **韭菜型散户** | 几万→几十万 | 低 | 追涨杀跌，看消息炒股，被所有资金类型收割 |
| **老股民型散户** | 几十万→几百万 | 中 | 有一定经验，迷恋技术分析，有体量无体系 |
| **大户/牛散** | 几百万→几千万 | 中高 | 可能有内幕渠道，可能做庄思路，跟风大资金 |

---

## 决策依据——每类散户的不同逻辑

### 韭菜型散户

| 决策驱动力 | 表现 |
|------|------|
| **价格本身** | 「涨了就觉得好，跌了就觉得坏」 |
| **消息面** | 群里说的、大 V 推的、龙虎榜看到的——全是二手信息 |
| **短线精灵** | 看到直线拉升 → 冲动追进 → 被量化收割 |
| **锚定效应** | 「已经跌了 30% 了，应该到底了吧」 |
| **社交媒体** | 看涨说涨、看跌说跌的自媒体最容易煽动他们 |

### 老股民型散户

| 决策驱动力 | 表现 |
|------|------|
| **技术分析** | 过度迷恋——画线、看量价背离、等均线回踩 |
| **经验直觉** | 有体量、有一些经验，但没有体系 |
| **路径依赖** | 过去赚钱的方法现在不灵了，但还在用 |
| **认知错位** | 用趋势方法分析游资票，用短线方法分析机构票 |

### 大户/牛散

| 决策驱动力 | 表现 |
|------|------|
| **内幕渠道** | 有重组消息、行业消息来源 |
| **庄家思路** | 选小盘标的，沟通上市公司，等风来 |
| **跟风大资金** | 跟着游资或机构方向走，但反应慢一拍 |
| **资金优势** | 钱是游资的好几倍，但没有游资的水平 |

## 行为特征

### 韭菜型的典型错误

杰哥总结了韭菜的标准画像：

1. **追涨杀跌不自知**——被情绪控制，完全不知道自己在做什么
2. **迷恋技术分析**——在游资票上看量价背离、等五日线
3. **过度依赖消息面**——盘中追各种直线（那是量化在诱多）
4. **定位混乱**——用机构的角度分析游资票，用游资的角度分析机构票
5. **主观判断的陋习**——在市场没迹象时提前买入，熬不住卖飞，过两天连板
6. **把短线做成长线**——被套了不服，加仓解套，越套越深
7. **意淫**——走弱了还在想会不会有资金来做双头

> 「拿着一个票就马上上技术分析、去追各种直线的——不用想了，你就是我口中说的又菜又爱玩的散户。」

### 老股民的典型局限

- 有经验，但经验来自已经被淘汰的市场环境
- 「过去散户占比高，游资一普及打法，散户就跟风模仿形成合力——这种模式已经失效了」
- 现在机构+量化占比提升，老股民的技术分析准确率越来越低

### 大户的尴尬

> 「有些大户的钱是游资的好几倍。但他不一定牛逼。」
- 钱多≠水平高
- 有内幕渠道≠会用内幕赚钱
- 做庄思路在现在的市场已经越来越难了

## 心理画像

### 韭菜型散户的核心心理

```
看到涨了 → FOMO（怕错过） → 追进去
追进去 → 回调 → 恐惧 → 割肉
割肉 → 又涨了 → FOMO 更强 → 追更高
追更高 → 真正见顶 → 深套 → 绝望
绝望 → 割在地板 → 反弹 → 循环
```

这是最稳定的心理模式——它的可预测性让其他所有资金类型都能从中获利。

### 为什么要自我认知

杰哥说散户最大的问题不是"技术不好"——是**不知道自己是谁**：

> 「你没搞清楚自己的定位。做短线就用短线角度分析，做趋势就用趋势角度分析，做科技大趋势就用机构角度分析。这节课就是给你们灌输这个概念。」

---

## 在食物链的位置

散户是整个 A 股生态的**底层能量来源**：

```
散户的情绪化交易
  │
  ├─→ 提供流动性 → 游资的对手盘
  ├─→ 提供波动 → 量化的套利空间
  ├─→ 恐慌割肉 → 机构的低位筹码
  ├─→ 追涨狂热 → 庄家的出货窗口
  └─→ 绝望离场 → 某队的抄底时机
```

**没有散户，就没有 A 股的整个短线生态。** 散户不是"应该被消灭的"——他们是生态系统不可替代的能量输入层。关键不是"不当散户"，而是**知道自己是散户，在这个生态里找到自己能赚钱的位置**。

---

## 如何脱离散户思维

### 铁律

1. **搞清楚自己的定位**：做短线 = 短线框架；做趋势 = 趋势框架。不要混。
2. **戒掉主观判断**：市场没迹象 → 不做。不要提前埋伏。
3. **戒掉意淫**：走弱了 → 认错。不要想"会不会有资金来做双头"。
4. **不看短线精灵的直线拉升**：「把自己变成盈利的对立面，你就去看直线拉升。」
5. **不追消息面上的直线**：「人力拼不过机器。消息出来 10:10 你 10:15 才看到——已经被量化收割完了。」
6. **做错了立刻认**：隔夜摁掉，不留恋。

### 进阶路径

```
韭菜型散户 → 认清自己的定位 → 只用一种框架
  → 学会判断资金类型 → 匹配分析方法
    → 学会预判别人的行为 → 从猎物变成猎手
      → 忘了自己是散户 → 你已经是游资了
```

## 实战案例

### 追涨科技被套的散户

> 杰哥说：「当有很多追涨杀跌的散户去问科技能不能追的时候，一旦出现游资的身影——前期涨得好的轮动上涨标的，通常会进入从加速到盘整的阶段。」

- 散户追进去 → 开始横盘或下跌
- 熬不住 → 在震荡中割肉
- 割完 → 开始拉涨
- 又追进去 → 又横盘
- 最终：赚了一点毛毛钱，甚至亏损
- **根本原因**：用短线的追涨思维去参与机构主导的趋势股

### 主观判断卖飞的散户

> 「在市场还没迹象的时候，你主观看好、提前买入、熬了几天卖飞、过两天连板。然后你会说：杰哥，这个票我早早就看好了，就是拿不住。」

这不是"拿不住"的问题——这是**主观判断**的问题。正确的是：不等市场确认信号 → 不买。而不是：提前买 → 熬不住 → 卖飞 → 懊恼。

> 「散户贴上一个准确的标签——叫做又菜又爱玩。这里无关钱多钱少的问题，只要你水平足够菜，你就是散户。」

> 「追涨杀跌容易被情绪控制，完全不自知。你不知道你是散户，你用机构的角度分析，又用游资的角度分析——没搞清楚自己的定位。」

> 「拿着一个票就马上上技术分析，去追各种直线的——不用想了，你就是我口中说又菜又爱玩的散户。」

> 「做错了一个标的，你根本不用思考明天，不用对它做什么预期。隔夜摁掉就行了，反正我做错了。」

> 「技术分析的尽头就是裸 K。只看量价——高手也只看量价。」
