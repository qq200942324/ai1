# 机构

> 「用短线的思维看 CPO，狗都不要；用机构的角度看，正是时候。」——杰哥

---

## 定义与内部分类

机构不是铁板一块。公募和私募是两种完全不同的物种：

| 维度 | 公募 | 私募（趋势型） | 私募（短线型） |
|------|------|------|------|
| **体量** | 百亿级，进出需要报备 | 十亿级，相对灵活 | 亿级，高度灵活 |
| **速度** | 最慢——建仓需数周 | 中等——可在流动性好的标的中快速进出 | 快——第二天就可以出来 |
| **核心驱动** | 基本面（景气度、估值、业绩拐点） | 行业拐点 + 趋势确认 | 情绪 + 题材 + 流动性 |
| **审美** | 好东西 + 便宜 | 好东西 + 趋势已起 | 热点 + 辨识度 |
| **龙虎榜** | 很少上，上了=加速期（警惕） | 偶尔上 | 经常上，席位背后可能是一伙游资 |
| **本质** | 趋势推动者 | 趋势跟随者 | 披着机构外衣的游资 |

---

## 决策依据

### 公募机构的三问框架

公募基金经理做决策时，脑子里只有三个问题：

1. **是不是好东西？** — 行业景气度、公司竞争力、业绩增速
2. **便不便宜？** — PE 分位、PEG、DCF 估值
3. **什么时候走？** — 是不是太贵了、交易是不是过热了

这三个问题跟短线没有任何关系。杰哥说：「在他们的眼里，不会过于看重短线上的得失。」

### 公募的进场时机

公募进场不是因为"涨了"——而是因为：

- **业绩驱动**：季报超预期 → 上调盈利预测 → 必配
- **行业拐点**：行业从低谷转向复苏 → 提前布局 → 越跌越买
- **流动性冰点**：市场极度冷清时 → 估值打到历史低位 → 逆向建仓

杰哥在 CPO 最低迷时说「正是时候」——这不是技术分析，是机构视角：「已经足够便宜了，而且后面起来一定是要通过科技去带人气的。」

### 私募的两面性

> 「有些短线大佬到后面也转去做私募，他们所控制的这一类型小私募，你觉得它会不会去做恒宝这种标的？」

私募的灵活在于：
- 可以做趋势（跟公募一样看基本面）
- 可以做短线（跟游资一样看情绪）
- 可以借机构席位规避监管（龙虎榜上看到的"机构席位"，背后可能是一伙游资）

## 交易特点

### 公募：最慢、最重、最可预测

| 特征 | 表现 |
|------|------|
| **买卖方式** | 分批建仓/减仓，单日买卖不超过成交量的 5-10% |
| **持仓周期** | 季度到年，不会因为一天的大跌就清仓 |
| **交易频率** | 低——调仓以季度为单位 |
| **涨停板** | 「不常涨停，更少会连板」 |
| **走势特征** | 轮动上涨，进二退一，沿均线爬升 |

### 私募：灵活的双面人

| 类型 | 交易行为 |
|------|---------|
| 趋势型 | 类似公募但更快——确认趋势后加仓，破位就减 |
| 短线型 | 类似游资——昨天买、今天卖，上龙虎榜也不怕 |

## 心理画像

**公募基金经理的心理**：
- 不急。体量决定了他急也没用。
- 怕的不是跌，是"买贵了"——买贵了会被领导和持有人问责。
- 怕的不是错过涨停，是"仓位不够"——在底部买少了才是真正的失误。
- 核心焦虑：排名。不是赚不赚钱，是比别人赚得多还是少。

**私募的心理**：
- 更在乎绝对收益（不像公募在乎相对排名）
- 更灵活但也更脆弱——客户赎回压力比公募大
- 短线型私募的心理本质上就是游资

## 在食物链的位置

```
某队托底 → 机构敢加仓 → 趋势确立 → 游资敢做情绪 → 散户追进 → 机构高位兑现给散户
```

机构是**趋势定价者**。没有机构的持续买入，趋势就只是反弹。机构不制造波动——游资和量化制造波动——机构利用波动来建仓和出货。

## 可预测性

| 事件 | 机构大概率行为 | 确定性 |
|------|------|:---:|
| 季报超预期 | 加仓 | ⭐⭐⭐⭐⭐ |
| 季报不及预期 | 减仓/清仓 | ⭐⭐⭐⭐⭐ |
| 估值到历史低位 | 越跌越买 | ⭐⭐⭐⭐ |
| 估值到历史高位 | 逐步兑现 | ⭐⭐⭐⭐ |
| 短线大涨 | 不追，等回调 | ⭐⭐⭐⭐ |
| 短线大跌 | 如果是错杀→加仓；如果是基本面恶化→减仓 | ⭐⭐⭐ |

机构是所有参与者中**可预测性最高的**——因为他们的决策框架是公开透明的（基本面模型），只是执行时机不确定。

## 如何利用机构的行为

### 可以做的

1. **机构必配的标的** → 在机构建仓前抢跑（需要大局观，如杰哥的「正是时候」）
2. **机构趋势已成** → 等回调到均线附近低吸，不要追涨
3. **没有大局观怎么办** → 杰哥说：「你要吃的应该是他回踩的那一下。你要不要上？」

### 不要做的

1. **不要追涨机构趋势股** → 「十追十一亏」
2. **不要用短线方法分析机构票** → 「用短线的思维看 CPO，狗都不要」
3. **不要在龙虎榜看到机构席位就激动** → 可能只是短线私募借的席位，第二天就跑了
4. 如果看到 CPO 大票上龙虎榜 + 大量游资席位 → 「警惕了，可能第二天就要铺盖了」

## 实战案例

### 案例一：CPO「正是时候」（2026 年 4-5 月）

杰哥在 CPO 跌得最惨、帕玛特和创新药在涨的时候，说「一定一定要开始关注这边了，因为这边已经足够便宜了」。

- **当时短线视角**：「弱的一批，狗都不要」
- **机构视角**：估值到历史低位 + 科技是带人气唯一方向 = 正是时候
- **结果**：新易盛从底部翻倍

### 案例二：赛力斯的四次连板（跨年度）

赛力斯是上千亿的大票，但在特定时期走出连板：

| 时间 | 背景 | 共同点 |
|------|------|------|
| 2022.04 | 上海解封、新能源车刺激 | 大周期背景 |
| 2023.09 | 华为汽车分支 | 大主线题材 |
| 2024.02 | 雪球爆仓后的普反 | 大盘重大拐点 |
| 2024.09 | 924 牛市 | 超大周期 |

**规律**：大市值票走连板的前提 = 超级大周期 + 市场极度活跃。平时不要指望大票连板。

### 案例三：恒宝——从短线到趋势的蜕变

前期：游资主导，连板凌厉，成交量小
后期：机构 + 散户接管，成交量上台阶，走成趋势

关键转折点：成交量明显上了一个台阶 ≠ 出货信号 → 参与者结构变了。
