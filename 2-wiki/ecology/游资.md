# 游资

> 「做短线票的关键就是随波逐流——当一根摇摆最快的墙头草。别人骂你是墙头草的时候，你可以把它当做是对你的称赞。」——杰哥

---

## 定义与内部分化

游资是 A 股最活跃、最受关注、也最容易被误解的群体。核心定义：**题材热点驱动、流动性敏感、以击鼓传花为核心盈利模式的专业短线交易者。**

但游资不是同质化的。杰哥把游资分成了两个时代：

| 维度 | 传统型游资 | 进化型游资（综合型） |
|------|------|------|
| **核心手法** | 纯粹接力/打板/连板高度判断 | 打板 + 趋势 + 波动 + 低吸多手法并行 |
| **参与标的** | 十几亿→几十亿小票 | 十几亿→几百亿，有赚钱效应就上 |
| **分析框架** | 板数→情绪周期→接力 | 资金类型判断→手法匹配→多维度博弈 |
| **适应力** | 被量化 + 机构化市场边缘化 | 随市场变化而变化，持续进化 |
| **代表** | 只会接力的老派游资 | 杰哥、老张（百亿级）等 |

> 「现在的游资已经进化了。他们时刻跟随着市场变化而变化，已经做的是综合型短线。而那些跟不上市场变化的人，也成为不了游资。」

---

## 决策依据

### 游资的三层博弈

游资不是在看"这个题材好不好"——是在看**别人会不会觉得这个题材好**。这是二阶博弈。

| 博弈层级 | 问题 | 对应的买卖动作 |
|:---:|------|------|
| 第一层 | 这个题材有想象力吗？ | 决定要不要关注 |
| 第二层 | 散户会追吗？潜在买盘有多少？ | 决定要不要点火 |
| 第三层 | 今天我买了，明天谁来接？ | 决定仓位大小和出场时机 |

杰哥原话：「时刻衡量潜在的空头跟多头力量的变化。」

### 核心买卖公式

**进场信号**（路转粉的开始）：
```
人气回落（关注度低） + 首板点火
→ 不关注的人 = 潜在买盘
→ 首板 = 路转粉的触发点
→ 次日有新买盘 = 溢价
```

**出场信号**（衰竭板）：
```
市场非常名牌的高潮 + 核心人气标封烂板 + 放出巨大成交量
→ 所有潜在买盘已经全部进场
→ 短期间内没有更多看多做多的资金
→ 次日无承接 = 必须离场
```

## 交易特点

### 手法全景

| 手法 | 适用场景 | 核心逻辑 |
|------|------|------|
| **打板** | 题材启动、分歧转一致 | 确认强度，拿到先手 |
| **半路** | 竞价超预期、盘中弱转强 | 比打板更早拿到筹码 |
| **低吸** | 恐慌砸盘后的筹码真空期 | 把自己当成量化，爆量下杀后抄底 |
| **接力** | 龙头确认后的加速段 | 击鼓传花，赚后手资金的钱 |
| **做 T** | 量化主导的高波动标的 | 冲高就砸，砸下来就吸 |
| **埋伏** | 题材预期阶段 | 有辨识度 + 筹码好 + 等风来 |

### 传统型 vs 进化型的标的偏好

| 维度 | 传统型参与 | 进化型参与 |
|------|:---:|:---:|
| 小市值（十几亿） | ✅ 核心战场 | ✅ 仍会参与 |
| 中市值（几十亿） | ✅ 最常见 | ✅ 最常见 |
| 大市值（百亿+） | ❌ 不碰 | ✅ 只要赚钱效应够强 |
| 趋势股 | ❌ 用短线思维分析（错） | ✅ 切换趋势框架参与 |

### 关键特征

- **灵活**：「油资买个 1000 万，你觉得他有多看好？人家只是练个手。」
- **不看重基本面**：「一本正经分析为什么 A 涨不过 B——误入歧途。」
- **看重流动性**：没有成交量，游资进不去也出不来
- **看重赚钱效应**：没有赚钱效应的市场，游资休息

## 心理画像

游资的心理状态是最复杂的——因为他们是**职业博弈者**：

1. **没有执念**：做错了 → 隔夜摁掉，不留恋。杰哥说：「做错了招商南油，根本不用思考明天，隔夜摁掉就行了。」
2. **没有主观**：「在市场没有迹象起来的时候，你主观看好它、提前买了、熬了几天卖飞、过两天连板——这是非常主观的行为。亏得最惨的人就是主观判断的陋习最严重的人。」
3. **极度理性**：不是在判断股票好坏，是在判断**人的行为**。游资的核心能力是预判散户 + 预判其他游资。
4. **反自我**：「你摇摆得越快，水平就越高。」

## 在食物链的位置

```
游资 —— 第三层·情绪引擎
  │
  ├─→ 点火制造赚钱效应
  │    └─→ 散户发现"涨了" → 追进
  │         └─→ 游资在散户高潮时出货（散户的买点 = 游资的卖点）
  │
  ├─→ 被量化利用
  │    └─→ 游资点火 → 量化监测到波动 → 放大波动套利
  │         └─→ 游资反过来利用量化制造的波动做 T
  │
  └─→ 与机构时间错位
       └─→ 同一标的：游资吃完情绪段 → 机构接棒趋势段
```

## 可预测性

游资的行为受两个因素驱动：**赚钱效应**和**流动性**。

| 市场状态 | 游资大概率行为 | 确定性 |
|------|------|:---:|
| 某队托底 + 赚钱效应出现 | 积极做多 | ⭐⭐⭐⭐ |
| 题材有持续性 + 消息不断 | 反复参与，多做几波 | ⭐⭐⭐⭐ |
| 亏钱效应扩散 | 大面积撤退 | ⭐⭐⭐⭐ |
| 量化主导 + 市场弱 | 收缩或休息 | ⭐⭐⭐ |
| 监管收紧 | 降仓位、降频率 | ⭐⭐⭐ |

游资的方向可以预判（题材来了会来，赚钱效应没了会走），但精确时机需要盯盘判断。

## 如何跟随游资

### 可以做的

- **识别游资点火信号**：人气回落后的首板 → 路转粉开始 → 可以参与
- **识别游资撤退信号**：高潮日核心标放巨量烂板 → 衰竭板 → 必须离场
- **用游资的视角分析短线标的**：只看筹码 + 情绪，不看基本面
- **做摇摆最快的墙头草**：不符合预期 → 立刻走，不犹豫

### 不要做的

- **不要用趋势方法分析游资票**：均线、估值、五日线对游资票没意义
- **不要提前埋伏游资票**（除非你有大局观确认风会来）
- **不要在游资大面积离场后还留守**：龙虎榜有滞后，等你看到已经晚了
- **不要在散户疯狂时追游资票**：游资正在把筹码交给追进来的散户

## 实战案例

### 恒宝——衰竭板 vs 路转粉的对比

| 阶段 | K 线形态 | 游资判断 | 操作 |
|------|------|------|:---:|
| 第一波高位 | 放巨量烂板 | 衰竭板——潜在买盘已耗尽 | 离场 |
| 回调后人气回落 | 缩量阴跌 | 关注度下降 = 潜在买盘积累 | 观察 |
| 第二波首板 | 缩量涨停 | 路转粉的开始——潜在买盘释放 | 进场 |

### 太和——被反推上板的离场信号

- **当日**：太和本来已经力竭，因为市场情绪好被反推上板
- **游资判断**：「下午这一笔拉升把所有犹豫要不要看多的人都追进去了。潜在买盘已经没有了。第二天基本上没有更多的买盘。」
- **操作**：当天就是卖点，不等五日线

### 招商南油——做错了就隔夜摁

- **杰哥**：盘中就知道自己做错了
- **操作**：「根本不用思考明天，不用对它做什么预期。隔夜摁掉就行了。」
- **启示**：游资没有执念。做错了不给自己找理由。
