# 量化

> 「量化的背后也是人为设定的模型。选股的是人，执行的是机器。你在选股上 90% 拼不过专业的人，在交易上 100% 拼不过机器。」

> **但**：「上完这一节课，希望你不再害怕量化。」

---

## 定义与构成

量化不是一支资金，而是一类**以统计模型驱动交易决策**的策略集合：

| 量化类型 | 策略逻辑 | 对市场的影响 |
|------|------|------|
| **趋势跟踪型** | 动量因子、均线突破 | 助涨——涨的时候追涨 |
| **统计套利型** | 价差回归、波动率均值回归 | 助跌——异常波动时反向交易 |
| **高频做市型** | 买卖价差套利 | 提供流动性，降低冲击成本 |
| **短线情绪型**（最新进化） | 引入短线情绪模型，监测热点+热度 | 最像游资的量化——会打板、会排撤 |
| **事件驱动型** | 爬虫抓取新闻 → 触发交易 | 比人类快几个数量级 |

---

## 决策依据

量化的决策是一个**信号→执行**的机械链条：

```
数据输入（价量/新闻/订单流）
  → 因子计算（动量/反转/波动率/情绪热度）
    → 信号触发（突破阈值→执行）
      → 算法执行（TWAP/VWAP/冰山指令）
```

量化不会问"这个股票好不好"——只问"这个信号的历史胜率是多少"。

## 交易特点

### 可识别的量化指纹

杰哥给出了精确的识别方法：

| 特征 | 量化标的 | 非量化标的 |
|------|:---:|:---:|
| **分时形态** | 直线拉升 → 直线跳水，反复出现 | 相对平滑的波动 |
| **对消息面反应** | 极其敏感，第一时间反应 | 有滞后，或根本无反应 |
| **龙虎榜** | 量化席位（东方财富拉萨等）频繁出现 | 游资席位或机构席位 |
| **流通盘偏好** | 几十亿的 20cm 小盘股是重灾区 | 大票一般量化占比低 |
| **做 T 行为** | 每天都在做 T，冲高就砸，砸下来就吸 | 单向买卖为主 |

### 典型的量化收割流程

```
Step 1: 监测到热点标的 + 人气上升
Step 2: 抢筹（展现为凌厉攻势，量化也在抢）
Step 3: 拿到筹码后 → 拉一根直线（2-3%）→ 散户看到拉升追进去
Step 4: 监测到追涨单涌入 → 砸底仓套利
Step 5: 砸到筹码稳定（单子变小、频率降低）→ 点一把火再拉上去
Step 6: 重复 Step 3-5，循环收割
```

### 一日游的量化

量化还有个特殊能力——**24 小时爬虫抓取消息**。所以手板特别快：

> 「量化打手板比你快。它是机器去收录消息的，你是人力。你家网不好，人家 10:10 弹出来的消息，你可能 10:15 才弹出来。」

第二天一个冲高就砸——如果题材没想象力 + 赚钱效应不好，一日游就出现了。

## 心理画像

量化没有"心理"——这既是它最大的优势，也是它最大的弱点：

**优势**：
- 不恐惧 → 该买的时候不会犹豫
- 不贪婪 → 该卖的时候不会留恋
- 不疲惫 → 24 小时监测，永远满状态

**弱点**：
- **模式固定** → 可以被预判和利用
- **模型同质化** → 多个量化模型同时触发 = 踩踏
- **在极端行情中会放大恐慌** → 2024.02 雪球爆仓就是量化集中平仓引发的

---

## 在食物链的位置

量化是**波动放大器**。它不制造方向，但放大方向：

```
行情强 → 量化抢筹助涨 → 游资更容易点火 → 散户更容易追
行情弱 → 量化触发离场 → 下跌加速 → 恐慌蔓延
行情中性 → 量化成为最活跃的资金 → 非量化参与者显得弱势
```

**量化的强弱不是绝对的**——杰哥的关键洞察：

> 「量化没有越来越强或越来越弱。它的强弱是有对比的——跟非量化参与者对比。只要非量化参与者很弱，就显得量化很强。市场弱的时候，你不想做，我不想做，机构也不想动——量化就是最强的时候。」

---

## 可预测性

量化是所有参与者中**微观可预测性最高**的：

| 场景 | 量化大概率行为 | 确定性 |
|------|------|:---:|
| 拉直线（2-3% 脉冲）后 | 砸盘套利 | ⭐⭐⭐⭐⭐ |
| 砸下去后筹码稳定 | 再点一把火拉上去 | ⭐⭐⭐⭐ |
| 消息面突发利好 | 第一时间抢筹（最快） | ⭐⭐⭐⭐⭐ |
| 消息持续性不足 | 第二天冲高就砸（一日游） | ⭐⭐⭐⭐ |
| 市场恐慌性下跌 | 多模型同时触发离场 | ⭐⭐⭐⭐ |
| 打板成功（封住） | 次日溢价较高 | ⭐⭐⭐ |

## 如何应对量化

### 杰哥的三条策略

**策略一：打不过就掀桌子**

> 「市场很弱、量化很强的时候——既然我打不过，就把桌子掀了。我不玩了。在球场也是啊，你打不过我，我把球给我，我要回家。」

当退潮期 + 量化主导时，最好的策略是空仓。

**策略二：把自己当成量化**

> 「你要参与的时候，首先判断标的里的量化多不多。如果量化说了算——把自己变成他。冲高就砸，砸下来就吸，做波动。」

具体做法：
- 打开分时图 → 看有没有直线拉升+直线跳水的规律
- 如果有 → 这就是量化标的 → 只做 T，不格局
- 如果在人气大票里（恒宝、中益达） → 量化说了不算 → 放心按正常短线逻辑做

**策略三：避开量化重灾区**

量化重灾区特征：
- 几十亿流通盘的 20cm 标的
- 分时经常出现无原因的直线脉冲
- 对消息面极其敏感（因为在爬虫监测中）
- 不是市场人气前列、但换手活跃

> 「找一个小一点的 20cm，怕的要死。你不知道它会不会突然触发离场点。你在恒宝里怕不怕量化砸？不怕。在中益达里怕不怕？不怕。因为量化说了不算。」

## 实战案例

### 通源/杰强——量化标的的波动操作

杰哥在讲通源和杰强时反复强调：「你是做波动的，你是做波动的，你是做波动的！」

- **操作**：高了抛，低了吸。不格局。
- **错误做法**：吸了等一直涨上去 → 砰一根大阴线腰斩
- **正确做法**：冲高就卖，恐慌就吸（但如果战争缓和那种向下的波动，就不吸了）

### 2024.02 雪球爆仓——量化集中平仓的极端案例

- **触发**：市场持续下跌 → 多个量化模型同时触发止损 → 集中卖出
- **放大**：量化卖出 → 更多融资盘被迫平仓 → 更进一步触发量化止损
- **终结**：某队出手，制造大拐点，打破负反馈
