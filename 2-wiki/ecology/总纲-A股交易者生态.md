## 核心生态原理

### 原理一：能量单向流动

整个生态的能量来源只有一个——**散户的追涨杀跌**。这不是贬低，是生态事实。

```
散户的情绪化交易 → 提供流动性 → 养活游资/量化 → 推动趋势 → 吸引机构 → 机构定价 → 吸引更多散户 → 循环
```

没有散户的情绪化，就没有游资的对手盘；没有游资的点火，就没有赚钱效应；没有赚钱效应，机构也不会进场。**生态的底层能量是不可或缺的。**

### 原理二：时间错位共生

同一只股票在不同阶段由不同类型的资金主导：

| 阶段 | 主导资金 | 分析方法 | 典型特征 |
|------|---------|---------|------|
| 启动 | 游资 | 板学/竞价/接力 | 连板凌厉，成交量小 |
| 加速 | 游资 + 量化 | 筹码博弈 | 成交量放大，分时波动剧烈 |
| 盘整 | 游资撤退，机构试探 | 趋势框架 | 横盘缩量，等方向 |
| 趋势延续 | 机构 + 散户 | 均线/估值/景气度 | 成交量上台阶，轮动上涨 |
| 见顶 | 散户主导，机构兑现 | 过热信号 | 龙虎榜游资密集出现 |

杰哥原话：「一个标的从底部走到头，没有人能从头吃到尾。你什么时候参与，反映了你是什么风格的交易者。」

### 原理三：分析框架必须匹配资金类型

> 「如果你用短线的思维去看趋势标的，你的思考就是错的，做什么决定怎么分析都是错的。」

| 资金主导 | 用这个框架 | 不要用这个框架 |
|---------|-----------|-------------|
| 游资主导 | 板数、竞价、接力、分歧转一致 | 均线、估值、五日线 |
| 机构主导 | 景气度、估值、行业拐点、是否便宜 | 连板预期、竞价强弱 |
| 量化主导 | 波动规律、分时脉冲、订单流监测 | 基本面、题材持续性 |
| 庄家主控 | 识别后避开 | 任何分析都没用——他说了算 |

### 原理四：相对强弱而非绝对强弱

量化的强弱不是绝对的——**行情强 → 量化弱，行情弱 → 量化强**。这不是量化本身变强了，是非量化参与者变弱了。

同样的逻辑适用于所有参与者：游资在震荡市更强，机构在趋势市更强，某队在恐慌市更强。**没有永远强势的参与者，只有适合当前市场阶段的参与者。**

### 原理五：博弈层级 = 预期嵌套深度

A 股的核心博弈不是"我预判事件"，而是——

> 「我预判了你的预判。」—— [[预期与预期差]]

| 层级 | 博弈内容 | 对应参与者 |
|:---:|------|------|
| 第 0 层 | 事件本身 | 散户（只看新闻） |
| 第 1 层 | 市场对事件的预期 | 机构（定价预期） |
| 第 2 层 | 别人怎么预期市场的预期 | 游资（预判散户行为） |
| 第 3 层 | 在别人预判之前先行动 | 某队 / 顶级游资 |

你在第几层博弈，决定了你赚谁的钱。
