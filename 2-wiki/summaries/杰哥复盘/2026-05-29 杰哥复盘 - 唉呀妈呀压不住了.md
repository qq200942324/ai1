# 2026-05-29 杰哥复盘：唉呀妈呀压不住了

> 国产区科技爆杀带动整个盘面科技股往下调整，带动海外区同时调整。大趋势风格调整区的特点：看着整个板块指数已乏力，个别品种跌幅大，却容易被还在新高的品种蒙蔽。

---

## 核心判断

1. **科技大趋势下行确认**：国产区爆杀带动全盘，海外区也被拖累
2. **第三次强调高低切换**：消费/银行/保险/白酒动了，更多资金从高位科技出来做切换
3. **不看A杀**：虽然调整猛，先看一个反抽看质量。极小概率出终极利好支持再走一波分歧转一致
4. **退潮有节奏**：不是一蹴而就，是"期待→犹豫→逐渐消亡"

## 五个关键问题（杰哥自问自答）

### 1. 为什么科技大趋势下行，辽能也不行？
- 资金担心中东反弹周期见顶，后续即使反复磨也耗时间
- 辽能和这个周期有特殊关系，周期见顶→辽能不行
- 面对周末，防守资金会更多。反而其他电力不错

### 2. 为什么大科技趋势下行，信W也不行了？
- 再升直接按死 → 商航大幅承压
- 信W和再升都是商航在这个周期的代表 → 一起G
- 同理：担心这个周期的问题

### 3. 为什么利T还在死顶？
- 这个周期仅剩的、有足够地位的高位抱团
- 今年涨幅最大、最持续的算力核心代表
- 市场爆杀反而加强利T抱团（控不住异动是情理之中）
- 周期见顶一定会受影响的

### 4. 天地板出现+打板大面 → 情绪不应该回暖吗？
- **弹性极限理论**：这边杀得太厉害 → 整个市场情绪受影响 → 短线炒作情绪一样受影响
- 担心周期从此见顶 → 高位情绪标已高 → 不指望向上穿越
- 打出新题材争夺风格变化的人也要考虑：大科技开杀后也会反复修复构建顶部结构
- 情绪风格走强的合力并不大

### 5. 低位的消费/银行/保险/白酒都动了 → 开始切换了吗？
- **是的，第三次说了。** 这次明显有更多资金从高位科技出来做切换
- 如果科技震荡下行，它们就会震荡向上

## 高位技术分析教学（核心干货）

> **"低位看逻辑，高位看技术分析。有效的技术分析没有太多花里胡哨的东西。"**

### 科创芯片
- 分时：上午反弹放出早上以来最大天量柱 → 攻击均价线失败 → 用了这么大力都不行=今天搞不上去
- 日线：阴线数量和阳线数量相同，但阴线量能明显放大 → **高位震荡中的空方结构**
  - 25日反攻缩量 → 趋势高位缩量加速只能看震荡
  - 昨天缩量反抽小阳很弱 → 今天大概率放量阴K修正
- 周线：连续三根放量冲高回落K → 下周5周线上移趋缓 → 易形成周线级别调整
- 月线：放量长上影 → 最好修正一下

### 半导体设备
- 分时：上午反弹攻击均价线缩量 → 更废
- 日线：顶部大量阴K → 25日大阳反包未创新高+量远不及阴K → 只能看震荡
  - 周四缩量反抽小阳 → 今天缩量大阴修正（设备调整比芯片更明显）
  - 因为已连续杀跌到要反弹的情况了 → 所以今天杀跌可以缩量
- 周线：两根冲高回落长上影 → 缩量调整 → 比芯片稍健康
- 月线：大幅冲高回落+放大量 → 需要筹码交换

### 通信（海外区代表）
- 分时：早上反击均价线带量不多→失败；下午再尝试→无量→再次失败；下午持续堆量放量下跌
- 日线：还算健康，多空相对平衡，**海外区强于国产区**
- 周线：量需要缓一下修正
- 月线：K线很健康，但量太大

### 指数分析
- **上证50**：连续下跌后放量反抽阳K，但下跌结构没走完，短期不构成止跌
- **科创50**：和芯片/设备分析差不多——顶部反抽加速+空方区域+周四缩量反抽
- **上证指数**：下跌结构清晰，阴K还在放量，调整区间空方占优 → 调整未结束，未看到企稳
- **创业板50**：近期最大量是21日大阴K → 后续反弹全部缩量 → 昨天易中天带动的阳线也缩量 → 不好

> 科创板代表国产区，创业板代表海外区。海外区强于国产区，但两者细节都陆续出现问题。

## 新概念提取

### 弹性极限理论
当主线杀得太厉害时，整个市场情绪都受影响——即使短线炒作情绪也一样被拖累。高位情绪标已高的情况下，不指望能向上穿越。

### 高位空方结构识别
- 阴阳K数量相等但阴线量能明显放大 = 空方结构
- 缩量反抽/缩量加速 = 弱势信号 → 次日大概率放量阴K修正
- 周线连续冲高回落+放量 = 周线级别调整前兆

### 缩量反抽修正规律
趋势高位缩量反抽/缩量加速 → 次日大概率用放量K或阴线修正。这不是偶然，是高位空方结构下的必然。

### 分时均价线攻防
上午反弹用最大量柱攻击均价线失败 → 全天搞不上去。缩量攻击均价线 → 更弱。
