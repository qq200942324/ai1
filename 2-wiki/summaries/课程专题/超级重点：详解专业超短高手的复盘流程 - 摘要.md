# 超级重点：详解专业超短高手的复盘流程

> 字数：约 6000 字 | 自动生成于 2026-05-27

## 核心要点

- 复盘 = 5倍经验积累：同一天内，你详细复盘获取的经验是其他人的5倍，盘感来自长年累月的经验积累而非天赋感觉
- 复盘的两步基础操作：数据录入（自选分组更新：涨停板/连板/炸板/大回撤/断板/主线/趋势/300标的/模型选股）+ 制作复盘表格（情绪数据计算/连板形态/强弱分组/亏钱方向/反包方向/核心高标跟踪）
- 分析的六个维度：指数环境分析 -> 流动性分析 -> 行业概念炒作分析 -> 情绪判断 -> 周期判断 -> 龙虎榜/实盘大赛/消息面复盘
- 流动性铁律：极致缩量市场=快速轮动/杀一致/杀抱团/抢跑/绞肉机/承接差/降预期；流动性充足=各种玩法随意/容错率高/可支持多线并行
- 交易策略制定的核心：必须要有前提条件（在情绪好转+低位顶一字助攻的情况下低吸龙头、在情绪修复+高标分歧时接力梯队），不是瞎几把头铁冲

## 内容摘要

本文是柚子小杰杰最详细的复盘教学，堪称短线交易者的工作手册。复盘的意义被作者凝练为"5年如一日的详细复盘积累"——盘感不是凭感觉下注，而是长年累月的经验积累形成的可推演的直觉。

复盘流程分两大步：（1）数据录入——完成9个自选分组的更新（涨停板/连板/炸板/大回撤/断板/主线/趋势/300标的/模型选股），每个分组都有明确目的；（2）制作复盘表格——情绪数据（大面/大肉/大幅回撤/大幅回升）、板块涨停潮统计、连板形态记录（一字vs换手/放量vs缩量）、核心高标分时文字记录。

分析维度从大到小：指数环境（6个指数）-> 流动性（决定玩法模式）-> 行业概念自上而下分析（板块指数 -> 核心高标 -> 权重 -> 中位/低位强度，叠加核心分时看日内博弈）-> 情绪判断（微观1-3天判断+阶段性统计判断）-> 周期判断（最难的维度）。最终所有分析浓缩为"精选次日观察标的"和"制定有前提条件的交易策略"。
