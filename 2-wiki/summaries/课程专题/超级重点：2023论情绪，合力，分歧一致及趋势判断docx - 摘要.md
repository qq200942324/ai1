# 超级重点：2023论情绪，合力，分歧一致及趋势判断

> 字数：约 8000 字 | 自动生成于 2026-05-27

## 核心要点

- 情绪的本质：众多心理因素驱动 -> 通过交易者决策行为反映在股价上 -> 是人性产物、市场合力产物，不可量化
- 合力的核心：大家好才是真的好——做多效应场景（点火造势者/借势跟进者/后知后觉者/企稳反弹/核心封板）与做空场景（双头/关键破位/恐慌抛售/非核心高溢价抢跑）
- 情绪演化的完整过程：场外资金累积 -> 打出局部赚钱效应 -> 跟风推动上涨 -> 高潮（场外枯竭）-> 筹码交换动态平衡 -> 方向选择 -> 退潮
- 核心人气股四维定位：高度（累积高度）+ 号召力（带动性）+ 抗揍程度（不坑人）+ 引领板块分歧转一致
- 短线情绪判断的完整工具链：一字板数量/封单/方向 -> 连板溢价率容错率 -> 炸板/跌停/大回撤反馈 -> 大盘涨跌比 -> 人气核心反馈

## 内容摘要

本文是柚子小杰杰"超级重点"级别的情绪课文字版，内容与视频3（重点大课：情绪判断）高度对应。核心框架分为三大模块：（1）情绪与合力——情绪是场内外普遍的心理感受，合力是各路资金综合驱动下的结果；做多合力的五种场景和做空合力的五种场景；（2）情绪演化的七步完整过程——从场外资金累积到退潮崩溃，每一步都对应筹码换手之间的动态平衡；（3）实战情绪判断工具——从一字板方向、连板溢价率容错率、到人气核心反馈的完整分析链路。

核心论点贯穿全文：情绪不可量化，周期阶段划分只能参考辅助交易，真正指导交易的是本质的赚钱效应。就像内功与招式——招式能打出更高伤害，但根基是深厚的内功。当别人变招时你的招式毫无作用，但如果内功深厚，无招胜有招。

最经典的实战应用：大分歧日不打缩量一封（因为那是情绪的惯性上行，是场内获利盘的合力卖点）；分歧转一致的二封才是买点（因为想卖的人已经卖得差不多了）。以及情绪溢出信号的出现（边缘化老龙头涨停）= 即将分歧的预警。
