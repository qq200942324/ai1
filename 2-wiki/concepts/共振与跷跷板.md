# 共振与跷跷板

> **「共振出妖股，指数搭台题材唱戏。」**

---

## 共振

### 三层共振

| 层级 | 内容 |
|------|------|
| 天时 | 政策环境、宏观条件 |
| 地利 | 当前市场环境、炒作环境 |
| 人和 | 合力——大部分资金看好，大资金看好 |

### 三类共振关系

| 类型 | 含义 |
|------|------|
| 指数/情绪/赚钱效应 三方共振 | 形成新主线主升做多周期，最优交易窗口 |
| 个股与板块共振 | 龙头领先板块启动，板块联动配合龙头主升 |
| 题材内多类型资金共振 | 短线圈 + 趋势股 + 大市值中军同步走强 |

### 共振的过程

1. 大盘指数出现拐点（止跌/反弹/翻转）
2. 某个方向领涨全场、放量、带动大盘放量止跌
3. 中军表态（大市值涨停或趋势起涨）
4. 短线品种打出赚钱效应标杆
5. 模仿、跟风批量出现 → 主流确立

---

## 跷跷板效应

当市场存量资金有限时，**两个方向此消彼长**：

- A 方向强 → 资金从 B 方向流出
- B 方向调整到位 → A 方向资金回流
- 把握节奏的关键：判断哪个方向在「强更强」、哪个在「分歧转弱」

---

## 关键认知

- 共振出来的方向 **持续更久、高度更高、宽度更大**
- 共振不是简单的「指数涨 + 情绪好」，而是 **多方资金在同一方向的协同**
- 跷跷板是存量市场的常态，切换要果断
- **注意**：「切换要果断」和「不切低位」不矛盾——前者适用于主线确认结束+新方向有共振时；后者适用于主线未死、防御只是短期跷跷板时。判断标准：主线龙头是否全面负反馈。详见 [[#跷跷板案例七：科技 vs 防御——不切低位的实战选择]]

### 实战案例

**共振案例一：大港股份——半导体芯片共振出龙头（2022年8月2日-8月12日）**

三层共振表现：
- **天时**：指数经历赛道股一轮惨烈杀跌后到达冰点区域，市场极度渴望新方向（来源：[[1-sources/8月14日]]）
- **地利**：半导体板块ETF整体涨5-6%，北方华创大市值率先企稳修复，权重搭台
- **人和**：大港股份打出地天板识别度，华大九天冲高回落助攻，芯片半导体方向获得短线资金共识

共振过程：8月2日地天板打出标杆 → 8月3日二板竞价确认 → 8月4日三板对抗老周期获胜 → 半导体全面爆发，苏州固德"一呼百应"引领回流，通富微电、康强电子跟风批量涨停。

结果：大港股份从首板走到8板，完整主升做多周期。作者原话：「我们看的层次不一样——我们看到的是对抗，你们看到的是烂板。」（来源：[[1-sources/8月14日]]）

**共振案例二：竞业达——信创共振出换手总龙（2022年10月10日起）**

共振特征：
- **指数共振**：大盘到达4月上海封城低点附近，形成指数和情绪双共振探底，出现"早晨之星"转折K线（来源：[[1-sources/3.周复盘+捕抓赚钱效应-10.21]]）
- **板块共振**：10月12日国脉科技一字板提供板块卡位——"援军来了"，批量的信创涨停板形成洪流
- **赚钱效应共振**：竞业达负五开盘日内拉升至涨停，日内15个点的赚钱效应完全明牌

作者比喻：「竞业达尝试通过一己之力带头扭转这个亏钱效应——就像孙中山留学归来告诉你，我有可能推翻清政府的统治。」（来源：[[1-sources/3.周复盘+捕抓赚钱效应-10.21]]）

**共振案例三：历史经典共振案例（作者总结）**

作者在共振教程中列举的经典共振案例（来源：[[1-sources/[2025-03-23]绝密干货浅谈共振与跷跷板效应]]）：
1. **比亚迪/宁德企稳大涨** → 共振出中通客车+科信技术，新能源储能产业链爆发
2. **中国交建+浙江建投** 二波三波联动，疫情基建计划刺激经济
3. **中国软件先起两个涨停板** → 共振出竞业达，信创数字经济
4. **万科A快速杀跌后止跌** → 共振出天保基建，房地产维稳

关键规律：「每次共振出来的大主流，背后逻辑一定硬，权重一定会动，不管它是先起涨还是刚起涨还是超跌反弹，一定不会是缩量震荡。」

**跷跷板案例四：AI数据 vs 会议线的混沌轮动跷跷板**

在混沌快速轮动市场（2023年某阶段），一周内轮动题材可达7-8个。作者的操作策略：
- AI数据和会议线争夺1-2天，汽车、机器人、消费、环保自然轮不上
- 环保医药这类逆指数方向，只有在AI和会议线都垃圾、市场都垃圾的情况下才会涨
- 具体操作：先选最强方向，再选受它影响最明显的跷跷板对手（识别度高、人气高的），去做次日的策略

**跷跷板案例五：盟固利 vs 正和生态——矛盾点做T（2023年8月）**

周四：全部题材都不行的情况下，盟固利有望跟新能源走强，但下午环保蹭上核污水概念，正和生态拉出准地天板，盟固利瞬间走弱回落。

周五：正和生态直接被按掉，盟固利低开——自然就是做T买入点。做T卖出点：正和生态承接起来而盟固利分时拐头向下的时候。

这个案例展示了"矛盾点"在实战中的运用——用于做T、做预期差低吸，一般用于混沌轮动市场而非主升阶段。
（来源：[[1-sources/[2025-03-23]绝密干货浅谈共振与跷跷板效应]]）

**跷跷板案例六：切换——从趋势到连板再到趋势的风格跷跷板（2024-2025）**

作者在0223第九节——关于切换中详细阐述（来源：[[1-sources/0223第九节——关于切换]]）：
- 2023年底从短线风格切换到趋势风格（人工智能大周期），需要提前准备
- "在23年整个行情里面，我感悟到更深的东西——持筹者和持币者的心理博弈"
- 短线风格和趋势风格之间存在跷跷板：当两者都有机会时，通过分歧来测试谁会胜出
- 核心策略：连板和趋势都参与，手就不会冷，一旦市场选择了方向，去弱留强

### 三概念联动分析

**[[看山是山]] → [[共振与跷跷板]] → [[赚钱效应]] 的逻辑链：**

**一看山是山的大局观如何帮助识别共振和跷跷板？**

看山是山的核心是"客观看待市场"——"让市场告诉你哪个方向强，不要自己先入为主"（来源：[[1-sources/1.2023绝密看山是山]]）。当指数拐点出现、某个方向放量领涨全场时，看山是山的客观视角能第一时间识别这可能是共振的起点，而不是主观猜测哪个题材会走强。在跷跷板场景中，看山是山帮助判断"谁的直线拉得更快更狠"、"谁先回封"——这些最直观的强弱信号。

**二共振与跷跷板的判断如何指导赚钱效应的捕捉？**

作者明确指出：「共振出来的方向，赚钱效应持续更久、高度更高、宽度更大。」（来源：[[1-sources/[2025-03-23]绝密干货浅谈共振与跷跷板效应]]）这意味着：
- 共振确认后应积极推仓位，因为这是最优交易窗口
- 跷跷板判断帮助在混沌期做套利——抓住"日内最强者"的短暂赚钱效应，或在A/B两个方向之间切换
- 共振失败（第一天领涨，第二天证伪）则果断离场，不恋战

**三赚钱效应是共振的结果验证**

共振是原因，赚钱效应是结果。如作者所说：「我们去理解去研究共振的目的，不仅仅是买到一个连板的龙头，我们要买到的是赚钱效应的引领者。」共振出主流的标志就是"以点带面"——有品种打出标杆、引发模仿、批量跟风出现，这正是赚钱效应扩散的过程。

#### 跷跷板案例七：科技 vs 防御——"不切低位"的实战选择（2026年5月27日）

**背景**：科技主线经历两日调整（科创50 两日 -4.2%），白酒午后拉升，科技同时跳水——典型的跷跷板效应。市场上有声音喊"切低位老登红利"。

**盘面**：「昨天讲到低位的老登红利相关的板块是有异动的，今天下午白水再拉一下，当时的科技是开始了跳水的。」（来源：[[1-sources/嘲讽讲讲]]，第33-34行）

**杰哥的判断**：「这里是否需要切低位老登呢，我和过去两年一样，我即使降仓位做做高抛低吸或者科技内部轮动，题材段打一下，我也不想去切这些板块。」（来源：同上，第34行）

**理由**：跷跷板的另一端（防御/红利）只是短期资金避险的临时去处，并非新主线。切过去后一旦主线修复，追回来的代价更大。杰哥的策略是：**承认跷跷板存在，但不跟随——用降仓位+主线圈内轮动来应对。**

**验证**：当天科技内部仍有大量冲高红盘机会——「今天的也有部分的科技线表现还是不弱的，即使回落那些，盘中都给了充足的冲高红盘的时间而且不少喔，在存储高开带头冲的时候，是很舒服的高抛点。」（来源：同上，第35行）

**教训**：跷跷板的出现不等于需要换方向。跷跷板提供了两个信息：①当前主线在调整（需要降仓位）；②有资金在另一侧避险（但这不是新主线）。正确的应对是"主线内做T+降仓位"，而非"切换去防御"。这补充了跷跷板框架中"不跟随跷跷板"的实战策略——跷跷板告诉你要谨慎，不是告诉你要叛变。

#### 跷跷板案例八：辽能（情绪标）vs 国产科技——散装主升中的情绪-趋势跷跷板（2026年5月）

**背景**：散装主升大周期中，情绪标的（辽能）与趋势标的（国产科技/海外链）之间存在持续的跷跷板关系。杰哥通过辽能三波完整演示了如何利用这个跷跷板做节奏。

**跷跷板节奏**（来源：[[2-wiki/summaries/杰哥复盘/2026-05-28 杰哥复盘 - 易中天新高与三波龙头]]）：

| 阶段 | 辽能（情绪端） | 国产科技（趋势端） | 跷跷板方向 |
|------|:---:|:---:|:---:|
| 启动 | 🔥 辽能启动 | 趋势风格杀跌 | 情绪 ← 趋势（资金流出趋势进情绪） |
| 分歧 | 辽能分歧 | 趋势蓄势反弹 | 情绪 → 趋势（资金回流趋势） |
| 二波 | 辽能二波 | 国产链 5/5 暴涨 5/6 冲高回落 | 趋势 → 情绪（趋势透支后资金切情绪） |
| 调整 | 辽能调整 | 国产区设备猛攻 | 情绪 → 趋势（趋势强时情绪弱） |
| 三波 | 辽能三波 | 科技即将反弹 | 情绪标的借科技反弹前最后惯性上冲 |

**当天盘面细节**（5/28）：
> 「辽能的几次冲高失败和国产区的科技品种形成鲜明的跷跷板这些细节，都可以看出，今天下午也会有一个高点T出。」

**核心规律**：在散装主升大周期中，情绪标的和趋势标的不是竞争关系，而是**节奏互补**——趋势分歧时情绪标的获得资金，趋势回暖时情绪标的让位。掌握这个节奏，可以在两端都找到交易机会。

**操作启示**：
- 不要因为趋势好就忽视情绪标的——散装主升中两者共存
- 跷跷板的转折点往往在趋势"透支"时（如科创50 一天涨幅=上一次两天涨幅）
- 情绪三波的空间锚定比二波更短，跷跷板切换要更快

> 来源：[[1-sources/[2026-04-13]海峡封锁Plus那咋整啊]]、[[1-sources/[2026-04-30]你的寒王无限猖狂]]

科技内部存在稳定的轮动顺序，一年多来反复验证：

```
光模块（海外链）→ PCB链（跟在光后面）→ 半导体链（更吃风偏）→ 国产算力（数据中心/服务器/液冷）
```

海外链先动→国产随后。如果还不熟悉这个轮动顺序，就是认知不到位。

## 海外链与国产链的节奏规律及共识形成机制（R26 新增）⭐⭐

> 来源：[[1-sources/[2026-04-30]你的寒王无限猖狂]]、[[1-sources/[2026-05-06]暴涨后部分冲高回落]]

| 阶段 | 海外链 | 国产链 |
|------|--------|--------|
| 先行 | 领先止跌先涨（光模块为代表） | 跟在后面 |
| 爆发 | 涨幅相对温和（旭创60%） | 爆发力更高（寒王80%） |
| 节奏 | 两三天涨完的量 | 一天干完 |

**共识形成机制**：错失海外链的人形成共识"国产区不差，不可能不轮到"→共识快速形成→海量买盘涌入→国产区一天涨幅=海外链两三天。

## "同是识别度高位"理论（R27 新增）⭐⭐⭐

> 来源：[[1-sources/[2026-05-14]批量兑现深绿巨亏]]

**对跷跷板框架的重大修正。**

高位时趋势和情绪不再是简单的"A涨B跌"跷跷板关系。每个风格、每个分支的识别度靓仔都被踏空资金**同步眷顾**——"市场太好，各干各的，踏空资金都在怼，有啥怼啥"。一旦兑现来临，所有高位品种"因为兑现而一起杀分歧"。

| 传统跷跷板 | "同是识别度高位" |
|-----------|----------------|
| A涨B跌，此消彼长 | A和B同步被推动 |
| 资金在方向间切换 | 踏空资金无差别怼所有高位识别度品种 |
| 一方见顶=另一方机会 | 同步兑现=全面杀分歧 |

**判断标准**：当市场出现"踏空资金有啥怼啥"的阶段→识别这是高位加速末期→不能再用跷跷板思维→应以防御为主。

> 来源：[[1-sources/[2026-05-14]批量兑现深绿巨亏]]

#### 跷跷板案例九：第三次高低切换 — 防御跷跷板趋势性确认（2026年5月29日）

**背景**：科创50 -5.04%暴跌，科技全线溃退。消费/银行/保险/白酒全面启动。杰哥明确说这是**第三次**提这个切换。

**杰哥的判断**（来源：[[2-wiki/summaries/杰哥复盘/2026-05-29 杰哥复盘 - 唉呀妈呀压不住了]]）：
> 「是的，今天是我要说的第三次了，前两次我都有提，今天这一次，明显有更多的资金从高位科技出来去做了切换。正常情况，如果科技震荡下行，他们就会震荡向上。」

**与前两次的区别**：

| 维度 | 第一次（5/27） | 第二次（5/28） | 第三次（5/29） |
|------|:---:|:---:|:---:|
| 跷跷板强度 | 午后拉一下 | 上证50独跌（资金在科技） | **全面启动**（电力/消费/金融批量涨停） |
| 科技状态 | 调整中但仍有强势股 | 普涨修复 | **全面溃退（科创-5%）** |
| 资金规模 | 试探性 | 少量 | **大量资金从高位科技出来** |
| 趋势性 | 短暂避险 | 一日游 | **趋势性切换信号** |

**核心规律**：防御跷跷板第一次出现=警惕、第二次出现=确认趋势在走弱、**第三次出现且伴随科技暴跌=趋势性切换信号**。此时应认真考虑降仓防守，而非继续"主线内做T"。

**与案例七的关系**：案例七（5/27）的结论是"承认跷跷板但不跟随"——这在第一次是正确的。但到了第三次（5/29），跷跷板已从"短暂避险"升级为"趋势性切换"，策略应从"不跟随"调整为"降仓防守"。

## "旧人不去，新人不出"板块轮动生命周期（R27 新增）⭐⭐

> 来源：[[1-sources/[2026-05-26060255]爆杀]]

高度浓缩的板块轮动逻辑：

```
旧主线不死 → 新主线难立 → 新主线弱鸡（"出而不堪"）→ 资金回流旧主线
```

**判断风格切换是否真来了**：旧主线的龙头是否全面负反馈？新方向的领涨是否有持续赚钱效应？两个条件同时满足=真切换；任一不满足=假切换。
