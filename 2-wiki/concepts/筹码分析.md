# 筹码分析

> 筹码结构决定了一个标的能否被快速拉升，以及拉升过程中会遇到多大的抛压。

---

## 核心概念

| 概念 | 含义 |
|------|------|
| 筹码干净 | 没有大量套牢盘，拉升阻力小 |
| 筹码集中 | 持仓集中在少数资金手中，拉升效率高 |
| 套牢盘 | 上方被套的筹码，拉升到对应价位会形成抛压 |

---

## 筹码在实战中的应用

- **补涨为什么快**：筹码干净、没有历史套牢盘、方便急速拉升
- **龙头为什么需要换手**：一字板上来的筹码断层，需要开板换手消化
- **套牢区是天然阻力位**：拉升到密集成交区必有分歧

---

## 与情绪的关系

- 好筹码 + 差情绪 = 没人拉
- 差筹码 + 好情绪 = 拉升吃力
- **好筹码 + 好情绪 = 牛股标配**

## 筹码的本质与地位

> 来源：[[1-sources/9.实战筹码核心体系]]

筹码是**辅助逻辑的强有力工具**。买卖依据逻辑，而盘中进出时机的选择，很多时候可以靠筹码辅助。筹码的本质是**交换**——交换过程是"筹码分散-集中-分散"的变换。

筹码需要与周期、逻辑、盘口等配合使用。交易系统从来都是各种大小影响因素的集合体，**没有所谓的一劳永逸一招搞定**。

---

## 九大核心知识体系

### 一、筹码分布（峰与谷）

筹码运动的主要特征：**流动性和形态的互相转化**。

- **峰**：股价波动趋窄，多空力量平衡。分为密集峰和分散峰
- **谷**：波动加剧，多空力量被打破
- **单峰密集**：对走势起支撑/压制作用，可按位置分为底部单峰密集、中继单峰密集、顶部单峰密集
- **多峰密集**：越往上峰值越高、高位多低位少 → 滞涨、调整时间延长、不利信号
- 筹码峰与均线构成**双重支撑/压制**

核心原则：**下峰支撑/上峰压制**。跌破下峰 → 反弹受阻；有效突破上峰 → 回调有支撑。

### 二、筹码聚合（核心买点）

筹码运动是一个 **"聚合 → 发散 → 聚合"** 的循环过程。买股就是买重新聚合。而**只有主力主导的向上聚合**才有确定性最高的机会。

聚合分两种：
1. **巨量承接处**：买卖双方大量换手后短暂平衡 → 新资金带量向上拉板打破平衡
2. **区间走横后的方向性选择**：高位走横多空平衡 → 有效聚合打破平衡的节点

有效聚合的判断标准：
- **必须带量**过前高
- 突破平台最高点，为多方有效聚合
- 非板大阳收盘仍在最高点上方 → 有聚合倾向，尾盘可进
- 没过最高点的只能算修复，不是买点
- 聚合之后=首板性质，次日正常走发散吃 5%-10%，运气好连板

经典案例（来源：`[[1-sources/9.实战筹码核心体系]]`）：
- **君正集团**：一字板→爆量首日→3 天小平台（聚合土壤）→第 3 天带量吃掉前上影→突破，这是有效聚合买点
- **浙江建投**：高位平台整理→第一次放量长阳尝试聚合失败→第二次放量尝试再次失败（但缩量了→聚合更高效）→第三天涨停板，筹码完全聚合上攻=扫板买点
- **特力 A**：9 连板放巨量→2 天一字跌停→修复 K 线均未过最高点→第 6 天回封板带量过前高=有效聚合买入信号

### 三、筹码发散（避险价值）

筹码发散是向上聚合后的必然过程：
- **向上发散**：持股信号，趋势持续，不做停留
- **向下发散**：危险信号，要防止

分时盘口层面：筹码散了 = 空方大单合力强 → 封不住概率大。筹码松动的前兆——**宽震、波幅和承接不受限制**（瀑布杀、爆拉爆跌），锁不住筹码，反转随时出现。

### 四、筹码结构（大局观）

**先有结构才有量价变化。筹码结构是一种大局观**，量价变化观察结构的演化，再用盘口锁定买卖点。

结构划分法（ABC 三段式）：
- A 点：起始定点
- B 段：不断延伸（可分前中后段）
- C 点：变盘点，成为新结构的 A 点

**量价背离是变盘预警**：阳线减少、阴线增多、量能减小→买卖盘都减小→多空动能消耗殆尽→变盘节点。

嵌套结构产生**双重确认买点**——同时突破结构 1 上沿+叠加有效聚合=确定性更高的买点。

研究建议：选择 50 亿流通市值以上个股，盘子太小结构稳定度不够；指数作为最大盘，结构最趋于完美。

### 五、筹码承接位

承接是上涨的**土壤**。筹码成本价 = 分时上剔除竞价开盘量后，第二和第三次量峰对应的价格（均价线也是重要参照）。

核心判断法：
- 价格回落至筹码成本区时**不放量** → 卖方暂时枯竭，买方未看空 → **极致低吸点**，多方点火四两拨千斤
- 价格回落至筹码成本区时**放巨量** → 多杀多，最后止损触发，空方宣泄 → **最后出场点**，不是低吸点

一句话：**"从贪婪到恐惧，构成极限低吸的基础，再结合量能是否放大，判断是买点还是卖点。"**

日内支撑判定的前提：必须有更高维度（周期向上/逻辑硬）支撑，不能只看分时形态。

### 六、左侧（心理压力区）

左侧不仅是筹码角度，也是心理压力区。两大威胁：**前高套牢区**和**成交密集区**。

情绪好龙头地位强 → 左侧轻松越过；情绪一般或面临大级别左侧 → 获利盘+套牢盘共振，高风险。

**后浪比前浪高**=筹码聚合到后浪；**后浪没前浪高**=后续面临 2 个前高压力。

"面临两种左侧，实在不知道该不该出手时，就退而观察，先不做买入，等过去了再说。持股遇见，也可以先撤，通过了再进不迟。"

### 七、衰竭与汇聚

最好的启动节点：**空方衰竭 + 多方汇聚**。

- **空方暂时衰竭**：下跌后振幅减小、跌幅收敛、成交量萎缩 → 某位置被认可 → 多方汇聚兵力
- **多方暂时衰竭**：上涨巨量大阳线 → 多方极致，将暂时衰竭 → 看后续空方逆反程度

分时应用：打板看多空双方几次集结的力度，"一次比一次强成功率大很多"，多方气势一旦怂了就尴尬。

上涨中继和下跌中继的小平台都是多空短暂平衡的结果。

### 八、筹码健康程度

筹码健康两大标准：**集中度高 + 换手好**（不矛盾）。

- 离套牢盘远、潜伏盘少、高位充分换手、集中度高不发散 = 健康
- 换太急/量不够就上攻 = 不健康，一旦炸板难回封
- 没换手没成交量的秒板一笔打上去→除非逻辑至上，不建议打
- **量健康不健康，看前面的均量值，不能太大也不能太小，差幅不超过 1/2 算平量**

"新鲜血液进来，老筹码出去，才是健康。"

盘口与筹码背离时，**筹码优先级更高**。除非出现"非他不可的情绪聚焦核心"，筹码即使不好看也能被强大情绪和资金拉起来。

### 九、成交量与换手

- "成交量是反映买卖双方在这个价格区间内对股价的认可程度"
- 缩量不能作假 → 可以用来判断其他地方是否真实
- 开盘前 5 分钟量最真实；换手板在 10 点前较好，强势市场只做开盘 15 分钟内
- 一字板造成的筹码断层：向上无支撑，向下不费力，都是情绪产物，非筹码自然交换
- 同一标的类似位置的换手率有参考意义："差不多同样位置时，它上两次板多少换手，这次大概是跑不偏的"

## 获利比例与极端值

- 并非越高越好或越低越好，**注意极端值变化**
- 新高大阳线获利比例达 100%，大阳底部仅 35% → 一天内差幅大 → 一旦反转，获利砸盘猛烈
- 获利比例极低（<2%）+ 筹码峰下沿 + 止跌 → 98% 以上资金亏损 → 下跌动能充分释放 → 常出现低吸机会

## 如何学习筹码

> 来源：[[1-sources/9.实战筹码核心体系]]

1. 熟悉基本量价关系
2. 选取 50 亿以上市值个股，盖住后面走势，一天天推演趋势的延续与反转
3. 多拆解当天盘口：什么变化导致散化？承接位表现？盘口共性？
4. **核心心法**："筹码理解的最深处就是持筹者和持币者的心态。恐惧贪婪，贪婪恐惧，你怎么去把握这个心理，理解这块资金怎么想，筹码不是学算法，更多的还是多空心理。"
5. "总舵主当年 3 个月手绘 3000 张 K 线图，搜尽奇峰打草稿"

## 实战证据

### 原文引用

1. **"筹码是辅助逻辑的强有力工具。买卖依据逻辑，而盘中进出时机的选择，很多时候可以靠筹码辅助。对筹码理解的深，可以极大降低介入成本，也可以卖在一天中相对高点，而且买卖会更自信。"**——来源：`[[1-sources/9.实战筹码核心体系]]`

2. **"筹码的本质是交换，交换过程是筹码分散-集中-分散的变换。落实到盘面上，筹码时刻是处于高速交换的过程中，筹码策略的应用比其他策略也更需要动态思考。"**——来源：`[[1-sources/9.实战筹码核心体系]]`

3. **"没有哪个模式能离开成交量，成交量本身就是筹码分析的一部分。成交量是反映买卖双方在这个价格区间内对股价的认可程度，记住这句话我觉得可以包含一半的成交量要义了。"**——来源：`[[1-sources/9.实战筹码核心体系]]`

4. **"筹码理解的最深处就是持筹者和持币者的心态。如果你是套牢盘，什么时候会不卖？拉到这个位置，你会不会从看空到看多？怎样的 K 能锁住潜在抛筹，让恐惧变成贪婪，既回本还不想卖？"**——来源：`[[1-sources/9.实战筹码核心体系]]`

### 实战案例

- **凯美特气（2025-04-07~08 关税冲击期）**：极冰日（周一 3000 家跌停）时翘板+打板凯妹入场，周二竞价转强后凯妹高开给机会加仓。背后的筹码逻辑：极冰当日虽跌停但被翘开说明有资金承接→筹码在低位重新聚合→次日竞价转强（多方表态）→加仓推满。来源：`[[1-sources/[2025-04-08]加仓打满]]`

- **君正集团（历史案例，教程拆解）**：一字板延续→爆量首日巨量换手→放量后没有往下走，短暂平衡（空方没有真正有效聚集）→构筑 3 天小平台→第 3 天带量向上拉板→打破平衡→有效聚合买点。这是一个典型的"发散→聚合→发散"的筹码运动教学案例。来源：`[[1-sources/9.实战筹码核心体系]]`

- **浙江建投（历史案例，教程拆解）**：一波过后高位走横→第一次放量长阳尝试聚合并失败→第二次放量尝试再次失败但量缩→第三次涨停板完全聚合→扫板买点。这展示了三次尝试才成功的筹码聚合过程，说明有效聚合不是一蹴而就的，"两次失败后第三次成功的概率大增，因为筹码越来越集中"。来源：`[[1-sources/9.实战筹码核心体系]]`

- **特力 A 妖股（历史案例，教程拆解）**：9 连板放巨量（第 1 天）→2 天一字无量跌停（没人出得来）→修复 K 线的 2 天都在平台内没过高点（不是买点）→第 6 天带量回封板突破平台最高点=有效聚合，可以扫板。注意：修复缩量不能打，只有过前高的带量回封才可以。来源：`[[1-sources/9.实战筹码核心体系]]`

- **寒武纪/科创板 ETF（2025-08-22）**：作者判断国产替代芯片方向后满仓出击。"即使买不到核心票，也可以去配 ETF。ETF 都能涨停。"筹码层面，主线方向确定后，整个方向的筹码都会受益，不纠结单票的筹码问题。来源：`[[1-sources/[2025-08-22]满仓涨停]]`

### 跨概念关联

- 与 [[情绪周期]] 的联动：筹码再好 + 情绪向下 = 没人拉。筹码需要与周期配合："一些偏离主线的分支题材，一直被主线压着，就算筹码结构好，也比较难给溢价。一些筹码很强的个股，也不能逆周期去硬怼。"（来源：`[[1-sources/9.实战筹码核心体系]]`）

- 与 [[龙头认知]] 的联动：筹码松动的预警（分时宽震、波幅不受限制、瀑布杀/爆拉爆跌）是龙见顶的前兆——"几乎每个龙头见顶之前，都会出现筹码松动的前兆，锁不住筹码，反转随时出现。"（来源：`[[1-sources/9.实战筹码核心体系]]`）

- 与 [[买卖点]] 的联动：筹码分析直接服务于低吸和打板点位选择——"如果你介入成本够低，也更能面对波动。多去理解筹码的承接，有助于寻找这种安全的介入点。"分时承接位不放量=低吸点，放量=出场点。（来源：`[[1-sources/9.实战筹码核心体系]]`）

- 与 [[分歧与一致]] 的联动：筹码聚合=分歧转一致的过程。分歧时筹码发散，一致时筹码聚合。有效聚合就是分歧后再次达成一致的技术确认。（来源：`[[1-sources/9.实战筹码核心体系]]`）

- 与 [[交易系统]] 的联动：筹码分析是交易系统进场时机判断中的重要一环——"筹码也好，盘口也好，所有的一切，都是为了在这个不确定的市场寻找确定性。稳定盈利就是你要把很多诱惑放弃，只做确定性最强的那个点。"（来源：`[[1-sources/9.实战筹码核心体系]]`）
