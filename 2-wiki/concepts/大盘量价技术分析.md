# 大盘量价技术分析

> 短线交易者的"大盘体检表"。不看大盘做个股=蒙着眼睛开车。本页提供一套完整的短线大盘判断框架。

---

## 盘面结构四维

每天开盘前和上午，用四个维度快速判断大盘状态：

| 维度 | 判断内容 | 信号含义 |
|:---:|------|------|
| ① | **权重放量才能进攻** | 权重无量拉升 = 假进攻 → 不追 |
| ② | **指数分工** | 上证50（权重/防御）= 护盘方向；科创50/创业板（科技）= 进攻方向 |
| ③ | **上午进攻比下午靠谱** | 上午放量拉升 = 真进攻；下午偷鸡拉升 = 做图 |
| ④ | **领涨不能是套牢盘题材** | 套牢盘小票领涨 = 超跌反弹非真行情；核心方向领涨 = 真行情 |

## 止跌六要素

判断大盘是否见底，需要六个要素交叉验证：

### ① K线底分型/顶分型

最基础的趋势转折信号：
- **底分型**：三根K线→中间最低→左右两边高于中间→下跌趋势可能转折
- **顶分型**：三根K线→中间最高→左右两边低于中间→上涨趋势可能转折

### ② 多级别联立

不同时间级别要互相验证：

```
5分钟 → 60分钟 → 日线 → 周线
 (日内)   (短线)   (中线)   (长线)
```

- 小级别服从大级别
- 所有级别同向共振 = 最强信号
- 大级别向上 + 小级别调整 = 低吸机会

### ③ 基准量

用 5日均量线和 10日均量线作为当日量能的对比基准：

| 量能对比 | 含义 |
|---------|------|
| 当日量 > 5日均量 | 放量，资金积极参与 |
| 当日量 ≈ 5日均量 | 正常量，存量博弈 |
| 当日量 < 5日均量 | 缩量，资金观望 |

**实战**：缩量阳线不可靠——缩量意味着没有足够外力打破震荡格局。

### ④ 开盘5分钟量能

开盘5分钟的量能可以**预判全日基调**：

| 5分钟量对比基准量 | 全日预判 |
|:---:|------|
| 显著 > 基准量 | 全天大概率放量、方向明确 |
| ≈ 基准量 | 震荡为主，方向需盘中确认 |
| 显著 < 基准量 | 全天大概率缩量、无方向 |

### ⑤ 均量线金叉/死叉

5日均量线与10日均量线的交叉信号：

| 信号 | 含义 |
|------|------|
| **金叉**（5日上穿10日） | 非盘即涨——要么横盘要么上涨 |
| **死叉**（5日下穿10日） | 非盘即跌——要么横盘要么下跌 |

**核心**：金叉不一定涨，但**死叉时做多胜率极低**。

### ⑥ 大阳大阴的三级支撑压力

放量大阳线/大阴线的 K 线实体部分提供三级支撑/压力：

```
大阳线实体
  ├── 实顶（收盘价）  ← 最强支撑
  ├── 实体 1/2 位      ← 正常回踩位
  └── 实底（开盘价）  ← 最后防线
```

| 回踩位置 | 支撑强度 | 操作 |
|---------|:---:|------|
| 实顶之上 | 最强 | 可积极做多 |
| 1/2 位附近 | 正常 | 谨慎持有 |
| 跌破实底 | 支撑失效 | 止损/减仓 |

## 综合判断矩阵

| 止跌要素满足数 | 大盘状态 | 操作 |
|:---:|------|------|
| 5-6 个 | 底部确认 | 积极做多 |
| 3-4 个 | 止跌形成中 | 试探性建仓 |
| 1-2 个 | 还未止跌 | 观望 |
| 0 个 | 继续下跌 | 空仓/轻仓 |

## 底部量能铁律（R30 新增）⭐⭐

> 来源：[[1-sources/短线常用的技大盘分析2]]

> 「**有量才是底，无量不是底。**」

| 量能状态 | 信号含义 | 操作 |
|---------|------|------|
| **底部爆量猛跌** | 做空动能集中释放→**衰竭信号** | 抄底机会 |
| **底部缩量阴跌** | 做空动能未释放→没到底 | 不抄底 |
| **放量阳线** | 增量资金进场 | 见底确认，可积极参与 |
| **缩量阳线** | 无增量，存量博弈 | 不可靠，观望 |

补充前面的止跌六要素：量能要素中加上「底部爆量猛跌是做空动能衰竭的抄底信号」。

## 领涨品种三不能（R30 新增）⭐

> 来源：[[1-sources/短线常用的技大盘分析2]]

有效的领涨品种必须通过以下三个筛选：

| 条件 | 原因 | 反面案例 |
|------|------|---------|
| **不能是大量套牢盘品种** | 套牢盘=拉升就被砸 | 上方筹码密集区的反弹 |
| **不能是题材小票** | 带不动大市值、无法形成板块效应 | 小票独自涨停无人跟随 |
| **不能是护盘性质的拉金融** | 除非牛市进攻属性，否则拉金融=假进攻 | 缩量拉银行/保险 |

> 「权重先行+小票进攻=有效上涨；小票强大票弱=根基不稳、无法持久。」

## 高位空方结构识别（R30 新增）⭐⭐

> 来源：[[2-wiki/summaries/杰哥复盘/2026-05-29 杰哥复盘 - 唉呀妈呀压不住了]]

> **「低位看逻辑，高位看技术分析。有效的技术分析没有太多花里胡哨的东西。」**

趋势高位时，技术分析的有效性远超基本面逻辑——因为高位定价已充分反映已知信息，方向由筹码博弈决定。

### 日线空方结构

| 信号 | 具体表现 | 含义 |
|------|---------|------|
| **阴阳K数量相等但阴线量能放大** | 多空博弈激烈，但空方每次出手的力度更大 | 空方占优 |
| **缩量反抽小阳** | 阳线缩量 → 只是卖压暂缓而非买盘回归 | 弱势信号 |
| **缩量加速** | 趋势高位阳线缩量加速 → 只能看震荡 | 非主升信号 |

**核心规律**：趋势高位缩量反抽/缩量加速 → **次日大概率用放量K或阴线来修正**。这不是偶然，是高位空方结构下的必然。

### 分时均价线攻防

> 「上午反弹放出早上以来最大天量柱去向上反攻均价线——失败。用了这么大力都不行，代表着今天基本搞不上去。」

| 分时信号 | 含义 | 全天预判 |
|---------|------|:---:|
| 放量攻击均价线→失败 | 多头用了最大力气也攻不上去 | 今天搞不上去 |
| 缩量攻击均价线 | 更弱，多头连力气都没出 | 全天弱势 |
| 下午无量尝试突破早盘高点→失败 | 多头第二次尝试，力气更小 | 尾盘大概率走弱 |
| 下午持续堆量放量下跌 | 空方发力，恐慌性抛售 | 次日大概率低开 |

### 周线级别调整前兆

| 信号 | 含义 |
|------|------|
| 连续三根周K冲高回落+放量 | 每次冲高都有人兑现，量能放大说明兑现力度在增强 |
| 本周放量 + 下周5周线上移趋缓 | 支撑减弱 + 空方增强 = 周线级别调整 |
| 两根冲高回落长上影+缩量调整 | 比前者稍健康（缩量说明抛压在减少），但仍需修正 |

### 月线量价

| 信号 | 含义 |
|------|------|
| 放量长上影月K | 大量资金在高位兑现，需筹码交换 |
| 放量大幅冲高回落 | 量能放大程度越大 → 需要交换的筹码越多 → 调整时间越长 |

### 综合判断框架

```
分时均价线攻防 → 预判当日方向
     ↓
日线多空结构 → 判断短期（1-3天）趋势
     ↓
周线量价关系 → 判断中线（1-3周）方向
     ↓
月线量价 → 判断长线（月度）趋势
```

> **「不好的细节陆续积累，会影响多空变化。所有的判断都是细节累积起来的，哪有什么我猜他要跌了。」**
