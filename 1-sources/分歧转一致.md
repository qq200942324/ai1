# **一篇文章讲明白"分歧转一致"的含义，掌握短线情绪变化本质核心**

今天讲一些轻松好懂的内容，关注我的朋友大部分都是做短线的。

肯定有听说过**买在分歧，卖在一致；分歧转一致，分化转分歧**之类的术语，刚接触短线的朋友肯定是搞得一头雾水。今天我就给大家普及一下短线的基础
什么叫做"分化， 分歧
，一致"。文章一定要看到最后，新手朋友学完一定大有收获，老手朋友学完也一定会有新的理解。

**分化，分歧，一致。其实是资金的一种多空心理博弈的体现，代表着市场大部分人的想法和行为。**

**一致**

**一致就是大家的想法相同，一致看多或者一致看空**，这个可以是个股，可以是概念题材，可以是一个大方向，也可以是整个大盘。

一般我们习惯用法就是分歧转一致，**这里的一致，就是一致性看多的想法和行为**。

表现在个股身上，最明显的特征就是**缩量上涨**。还有一种就是经历过**调整后继续向上涨**

**分歧**

刚说完一致的含义，那么分歧又代表什么呢？

**分歧就是大家想法不一样，有的看跌有的看空，一般是看空的力量逐渐大于看多的力量**。个股上量能会放大，成交量的放大，代表的是卖的人很多，买的人也很多，这里就是代表着大家的想法完全不一样，**分歧很大最后变现出**，**放量滞涨，或者转跌**。

**分歧到了极致就是情绪的冰点，一致到极致就是情绪的高潮。**

**分化**

那么分化又是什么意思呢？他的性质跟分歧和一致不一样，更多是一种两者当中的表现。

A B C D
四个股票。一开始大家一致看涨，所以abcd一起普涨，也有可能是一起涨停。然后第二天，有的人就开始认为，A更好，B也相对较好，那么去买A和

B的人就更多了，结果就是**CD相对AB，就涨得没这么好**了。

**当分化到极致的时候，有的票涨停，有的票大跌，大家都看出已经有问题了，那么正常来说后面就是开始分歧的时候了。**

下面我用一个实例给大家讲解一下，去年的芯片连板龙头大港

1\.
首先在大盘调整，短线情绪集体低迷的时候，我们看到他用地天板的方式，打出了自己
的识别度，**带领了赚钱效应**。

1.  次日继续涨停**延续赚钱效应**，然后带领了芯片成为这个时期的主流。

2.  经历了加速之后，迎来了**第一次分歧**，大港他是龙头，扛住了第一次分歧，在第四天**分歧转一致**，这个时候很多龙头选手看到这一点就会过来跟进了。

3.  第四天第五天**连续缩量加速**

4.  [第六天迎来了**第二次分歧**，也是放出了相对应较大的量能。]{.mark}

5.  结果是又抗住了一次分歧，在第七天**再次缩量加速**。

6.  [来到了第八天的分歧，已经是起涨以来的**第三次分歧**了，这次并没有转一致，而是开始了一个**盘顶**的动作，盘顶完也就**结束**了。]{.mark}

在大港上涨途中，各种的跟风股，有的长得好，有的涨得没这么好，这个就叫做分化。**当一个集体内个体差距悬殊，人心就开始不齐了，后面就是接分歧了。**我们通过龙头的走势，对分歧的判断就会更加直观。
