**三重界\-\--看山是山，看山不是山，看山还是山**

**\
宋代禅宗大师青原行思提出参禅的三重境界：**

**参禅之初，看山是山，看水是水；**

**禅有悟时，看山不是山，看水不是水；**

**禅中彻悟，看山仍然山，看水仍然是水。\
\
人生第一重界：看山是山，看水是水。涉世之初，还怀着对这个世界的好奇与新鲜，对一切事物都用一种童真的眼光来看待，万事万物在我们的眼里都还原成本原，山就是山，水就是水，对许多事情懵懵懂懂，却固执地相信所见到就是最真实的，相信世界是按设定的规则不断运转，并对这些规则有种信徒般的崇拜，最终在现实里处处碰壁，从而对现实与世界产生了怀疑。\
\
人生第二重界：看山不是山，看水不是是水。红尘之中有太多的诱惑，在虚伪的面具后隐藏着太多的潜规则，看到的并不一定是真实的，一切如雾里看花，似真似幻，似真还假，山不是山，水不是水，很容易地我们在现实里迷失了方向，随之而来的是迷惑、彷徨、痛苦与挣扎，有的人就此沉沦在迷失的世界里，我们开始用心地去体会这个世界，对一切都多了一份理性与现实的思考，山不再是单纯意文上的山，水也不是单纯意义的水了。\
\
人生第三重界：看山是山，看水是水。这是一种洞察世事后的反璞归真，但不是每个人都能达到这一境界。人生的经历积累到一定程度，不断的反省，对世事、对自己的追求有了一个清晰的认识，认识到"世事一场大梦，人生几度秋凉"，知道自己追求的是什么，要放弃的是什么，这时，看山还是山，水还是水，只是这山这水，看在眼里，已有另一种内涵在内了。\
\
"人本是人，不必刻意去做人；世本是世，无须精心去处世"，这才是真正的做人与处世了。**

**（去年的生态）**

**我一般是从3-4开始做起来，首先我晚上回板今天的三板全部按照涨停的顺序排序起来，搞清楚他们的涨停原因和炒作题材。其实很多人都说什么三板定龙头，虽说是个谬论，但是多少由他的道理，因为能涨到3板题材和逻辑一般都不会太差。**

**如果没有明确的主线，我们可以用涨停的时间来区分他们的强弱。**

**如果是同一梯队出现多个同一题材的标的，那么我们也是看这三个票里面涨停时间靠前的为最强。板块效应出来了**

**如果不同的题材，涨停时间靠后的或者烂板的都要观察，因为在一个主线朦胧还没走出来之前的，中位接力的弱转强太常见了。[（因为有可能通过中位的放量分歧弱转强，自身的超预期带动所在的方向，成为是市场的主线，那么他自然就是龙头了）]{.mark}**

**复盘的排序和加深印象心里有底之后，那就看次日的集合竞价阶段了**

**集合竞价是市场留给你最后调整计划的时间，通过集合竞价进一步区分强弱**

**以同身位股接力PK来说，如果在没有外力影响的情况下，那竞价就应该按照昨天涨停时间先后依次由强到弱，最先涨停的竞价开最高，然后依次到最后涨停的竞价开最低，还有昨天断板的，就应该绿盘开，这是理想状态下应该有的表现，也是大家常说正常预期。**

**还有一些晋级版本的预期，例如扛过了分歧引领回封的，次日情绪好就是加速这种，后面在实盘中给大家一举例。**

**但市场肯定有外力影响推动，那就会出现一些不正常的竞价表现，我们可以通过竞价的异常情况，进一步来区分强弱。**

**以同身位股的接力PK来说，前一天最强的票，正常应该大幅高开，但竞价仅仅小高开或者平开，可能就是不及预期；昨天涨停靠后的票正常是小高开，但竞价大幅高开就是超预期，平开甚至绿盘就是不及预期；再比如昨天大烂板的票，正常很可能低开，高开就属于超预期，断板的票高开也是超预期表现**

**[（不是所有的烂板，尾盘板都不能打，有的时候，那是超级高手的买点）]{.mark}**

**真假弱转强：超预期高开的情况下，考虑主线分歧一致的预期，主线的健康程度和阶段，考虑整个大环境情绪，还有一点就是他的所在的题材相应的对手（别的题材）今天表现怎么样，最后才是看竞价有没有放量。**

**另外还有相对强弱，比如昨天最强连板股是平开，那如果昨天大烂板股或者断板股也是平开，就可以定义为昨天的弱势票，今天相对强；**

**反过来如果昨天最后排甚至断板的票都能高开，反而昨天最强的票也仅仅是小高开，那这只票就是相对弱(该强不强是微弱，没有很明显的情绪和主线强度支撑，可以考虑破0轴就砸，或者减仓)**

**此外还有竞价卡位，最明显的就是同题材股的卡位，板块龙二竞价比龙一更强，这些都可以通过竞价表现看出来，除此之外，盘后消息也是影响竞价的重要因素**

**集合竞价之后就是开盘，盘中对于强弱的判断，和竞价其实大同小异，只不过盘中是动态表现，强弱关系可能会随时发生变化，但其实盘中的强弱关系并不是一直都在变化，相反强弱变化的情况并不多见（通常见于情绪
不太好的时候，情绪好的时候大家都是拉直线，就是很直观看到谁的直线拉得更快更狠）**

**比如两只票A和B，昨天表现、今天竞价以及开盘表现都是A票强，那B票想在盘中翻盘反超的可能性其实很小，这也符合看山是山的逻辑，强和弱通常都是很直观的表现。**

**大家都知道我是一个打板选手，极少在竞价和半路买票。我把这个盘中动态强弱看做是一场比赛，不到最后不知道谁会最快冲线。所以我会以最终的结果为准谁先上板谁就是强。（以前的生态，情绪好的情绪正常的生态，不是这种内卷的高速轮动的状态）**

**既然看山是山拓展到看强是强、看弱是弱，那就不能不提去弱留强，看山是山与去弱留强其实就是原理与方法的关系，知行合一则是实践阶段，买票和卖票的思路其实是相通的，都是设定预期通过判断强于预期还是弱于预期，然后决定去留**

**当接触了龙头战法和情绪周期之后，就会想，为什么明明是卡位，明明是补涨，为什么就是长不起来还A杀，难道是情绪周期的问题？**

**想了一下不是很懂，就套一下，开始画阶段，开始个股叠加，用通骚操作后，自认为搞明白了（依旧不知道对还是错，只是主观觉得这个问题得到了解决）当你想明白这一次后，好的下一次也不一样了，下个周期低于预期了也不一样了，下个主线炒作因为各种原因也半路GG**

**这里不是说自下而上的思考方式不对，也是对的，但是思考的路径确实错的。**

**我们的市场，股价和走势反应的多重因素都计算进去后反映出来的客观事实，这个时刻最真实的定价。**

**所以我们只需要知道他强或者不强，弱还是不弱；大家想不想要，他是不是一直涨，回调都不带蹲下的；**

**那就够了，我们要的是搞钱，根据客观现实去做应对就好了，他涨了我要不要追要不要卖，他跌了我要不要吸要不要卖，仅此而已。**

**自上而下的思考方式**

1.  **市场能不能涨，哪个方向能涨？大票还是小票，蓝筹还是成长还是市梦率**

2.  **哪个风格：短线连板接力，还是反包接力；还是趋势，是短线趋势还是蓝筹大浪趋势。市场偏爱大盘还是小盘，流动性和市场偏好又有什么关系。**

3.  **市场运行的周期是一个怎样的周期，可能不能看到切换的预期，赚钱效应怎么样，有没有延续性等等**

4.  **哪个主线：这个很简单，选最强的。**

5.  **每次分歧，哪个分支最强，哪个持续性最好，哪个涨最猛。**

6.  **接下来就是做与不做的问题，怎么做的问题，这个就是要结合情绪的判断和周期的判断，推演出交易节奏。**
